# 选股分析框架

## 投资哲学

**格雷厄姆（好价格）**：安全边际足够大，判断错了也亏不多。

**费雪（好公司）**：GM 高且稳、护城河宽、FCF 强、管理层称职。烂公司再便宜也不碰。

**巴菲特（一次性麻烦）**：好公司平时不给好价格，只在遇到**暂时**困难时才给——困难会过去，公司还是那个公司。

三者缺一不可：
- 只有好价格 = 价值陷阱（便宜的烂公司）
- 只有好公司 = 买贵了
- 好公司 + 好价格但麻烦非一次性 = 结构性恶化
- 三者齐备 = **最佳买点**

## 概述

识别"因暂时困境被市场低估的优质大公司"。**不追高增长**——合理 PE 硬封顶 30x（见第五步），增速 >~22% 的公司一律按 30x 估，给不出安全边际就判"买贵了"，等回调。

> 本文件只写**稳定规则**。具体数字、案例、估值结论会过时，一律进 `financial_data/{STOCK}/analysis.{YYYY-MM-DD}.md`，不写入本文件。

## 数据获取

```bash
# CHEAP + SCORECARD + 原始财报合并输出；STOCK 换成实际代码
python3 -m fundamental.combine STOCK
```
用户说"分析 STOCK"时，先跑此命令拿数据再执行五步；也可直接粘贴输出。

## 五步框架

| 步骤 | 解决 | 内容 |
|------|------|------|
| 一 + 二 | **好价格**（格雷厄姆） | 价格粗筛 + 五年估值分位 |
| 三 | **好公司**（费雪） | 财报健康 |
| 四 | **一次性麻烦**（巴菲特） | 护城河 / 前瞻增长 / 管理层 / 消息面 |
| 五 | **安全边际**（出手纪律） | 用前瞻 g 估内在价值，要求折扣买入 |

漏斗：一步宽松入池 → 二三四步精筛 → 五步定价。估值公式放第五步，因它需要第四步产出的保守前瞻 g。

### 第一步：价格粗筛（满足任意 1 条入池）

猎场是"因暂时困境**下跌**的好公司"，门槛宽、以"跌幅或便宜"为主：
- 回撤：1年 >40%、2年 >60%、距 52 周低点 ≤15%
- 绝对估值：P/E TTM <15x、EV/EBITDA <10x、P/S <2.0x（净利率>25%放宽至 4.0x）
- 金融股：P/B <1.0 替代 EV/EBITDA
- 周期股：正常化 P/E <12x、P/B <1.5x、净现金/市值 >30%、或股息率 >5%

不在此处做成长调整估值——深跌成长股靠回撤入池，其公平估值留到第五步用前瞻 g 做。

交叉验证：EV/EBITDA <2x 但 P/E >10x = 单位混淆风险（常见于非美元 ADR）。EV/EBITDA 查两个独立源。

### 第二步：估值分位

实时查 5 年 P/E（或 P/S）分位，须 ≤30%。查询失败则暂停，严禁估算。

**先判是否周期股**（盈利随商品/资产价格剧烈摆动：矿业/能源/航运/半导体/航空/券商/加密交易所）。是 → P/E 分位会误导（底部利润低、P/E 高=买点；顶部 P/E 低=卖点），**不能拿 P/E 当贵/贱信号**；改用：中周期正常化 P/E（盈利取跨周期中枢）+ P/B 分位（P/E >70% + P/B <30% = 周期底部）+ 净现金/清算价值。**好价格大前提不变**——周期股也须在中周期口径下有安全边际（第五步），底部 ≠ 任意价格买。

展开（不许只写"≈Xth 通过"）：
- **自身区间表**：当前值 / 5 年高/低/中位 / 前瞻，标分位，附来源 URL
- **同业对比表**：列可比同业当前或前瞻倍数 + 性质，算 PYPL 折/溢价%；剔除性质不可比者（如卡组织 vs 支付处理商）
- **de-rating 陷阱**：历史高分位若来自成长泡沫期溢价，低分位有一部分是"高增长小盈利→正常盈利"的范式切换，不全是错杀，需打折看

与三/五步闭环：便宜的成因（错杀 or 真实恶化）要点明，呼应第三步财报与第五步 g。

### 第三步：财务健康（SCORECARD + 财报 heuristic）

**SCORECARD**（权重 BS 40% / CF 35% / 利润表 25%）：
- 伤口模式：3yr 高分 + TTM/5Q 低分 = 困境反转候选
- 区分情绪伤口（基本面完好、市场恐慌）vs 结构性伤口（真实竞争侵蚀）

**财报 heuristic**（基于 STATEMENTS 原始数据）：

利润表：
- 列近 5 季 Revenue / GM% / OpM% / Tax% / NM%，找趋势与异常季
- GM 连续下滑 = 定价权/成本恶化；SGA/R&D 突跳 = 收购或战略投入，需查明；税率异常会扭曲 EPS，需正常化

利润增长质量（降本 vs 增长）：
- OpInc 倍数暴涨但营收停滞时，桥接拆解：增量来自"营收增长"还是"砍费用"
- 砍费用是一次性台阶（砍到位即止），营收增长才是可持续斜率——"拧毛巾 vs 接新水"
- 低基数放大倍数：OpM 从极低位爬升时任何改善都显倍数级，看绝对 OpM 是否已成熟
- 警惕"利润表满分掩盖营收引擎熄火"

GAAP 异常：
- Op Income 与 NI 背离 = 非经营项主导利润
- 识别一次性项（重组/减值/诉讼/处置）算正常化 EPS
- Other Income 大额波动 = 投资损益或 FX，不反映经营

资产负债表 & 现金流：
- Goodwill 跳升 = 收购，查标的对价
- OCF/NI <1 = 利润含金量低；SBC > NI 时用 `FCF − SBC` 看真实现金创造
- CapEx 急增可能吞噬 FCF；回购骤降 = 管理层优先级转移
- 追踪 diluted shares：回购是真缩股还是仅对冲 SBC
- **现金特别充裕必查来源**（决定它做安全边际锚时算不算数）：留存收益（最优，真赚的）> 发股（真但历史稀释）> 发债（净现金口径已扣，但杠杆隐患）；**负营运资本浮存**（应付膨胀撑起的现金）随销量反向回流、非永久，作锚要打折

### 第四步：增长前瞻、护城河、管理层、消息面

实时联网搜索（标 URL + 日期），从稳定到易变：

**增长前瞻**（产出第五步的 g）：
- 历史：3 年营收 CAGR + 近期季度 YoY（加速/平稳/放缓）
- 前瞻：分析师 3-5 年营收 CAGR 共识、管理层指引
- 质量：护城河能否支撑？竞争是否侵蚀？增长一次性（基数/恢复/并购）还是可持续？
- **产出前瞻可持续增速**：取历史与前瞻的保守者；近期显著低于 3 年 CAGR（放缓）则用近期

**护城河**：壁垒类型（网络效应/切换成本/规模/品牌/牌照）、竞争格局与份额趋势、威胁（新进入者/技术替代/监管）

**管理层**（记 fact 便于跨期追踪）：CEO/CFO 任期与背景、内部人增减持（金额+价格）、资本配置（回购时机/M&A 成败/SBC 趋势）、指引 vs 实际、治理风险（空缺/换帅/VIE）

**消息面**：近 3 月核心新闻（诉讼/回购/评级）、行业宏观影响

输出"熊市逻辑"与"牛市逻辑"。

### 第五步：安全边际（估内在价值，定出手价）

一般/成长股（有正的正常化 EPS）。**全程股价口径**：`正常化 EPS = 正常化 NI ÷ 稀释股数`，股数用收入表 `Diluted Average Shares`（已是股价口径，ADR 已换算为每 ADS）；别用 BS `Ordinary Shares Number`（原始普通股，ADR 差 2~4 倍）。MCap÷现价 仅交叉验。链条：EPS → g → 合理 PE → 合理价 → 安全边际% → 满仓目标。

```
正常化 EPS = 正常化 NI ÷ 股数          # 利润被一次性麻烦压低时,用恢复后正常 margin 的 EPS
g        = 第四步产出的保守前瞻增速     # 不再砍半(保守性已在第四步做);g≥~22% 基本顶到上限
合理 PE   = min(8.5 + g, 30)          # 价值框架硬封顶 30x:>30x 无吸引力,不追高增长
合理价    = 正常化 EPS × 合理 PE
安全边际  = 1 − 现价 / 合理价
满仓目标  = 合理价 × 折扣系数           # 折扣随【质量 × 麻烦确定性】滑动(见下)
```

**质量判定**（伟大 vs 平庸，据第三步 GM/NM/FCF + 第四步护城河）：伟大 = GM 高且稳 + NM 高 + 护城河宽 + FCF 强（yield 高、`FCF−SBC>0`）+ 真缩股；平庸 = GM 薄/下滑、护城河弱、FCF 平庸或靠浮存。

**满仓折扣系数**——质量越硬、麻烦越确定一次性，越敢接近合理价买（巴菲特"合理价买伟大公司"）；麻烦越大/越存疑，越要"四毛买一块"：

| 质量 × 麻烦 | 折扣 | 系数 |
|------|------|------|
| 伟大 + 麻烦明确一次性 | 25% | ×0.75 |
| 好公司 + 一般麻烦 | 33% | ×0.67 |
| 平庸 / 麻烦存疑 | 40% | ×0.60 |
| 重麻烦 / 结构存疑（赌反转） | 50-60% | ×0.40~0.50 |

仓位分档（10 只 × 10% 等权）：
- 现价 ≤ 满仓目标（合理价 × 折扣系数）→ 安全边际充足，分批建满仓
- 现价在 满仓目标 ~ 合理价 之间 → 不足，先小仓 / 观察
- 现价 > 合理价 → 买贵了，不出手

护栏：
- **回购不进 g**：g 用业务/净利润增长，不含回购驱动的 EPS 增长。回购价值体现在"低价回购=安全边际"+"缩股逐年抬升 EPS 基数"，不抬高合理 PE。净利停滞靠缩股撑 EPS 的，g 取业务增长（可能为 0）
- **g 质量**：`FCF − SBC > 0` 且 3 年增长。烧钱式增长（FCF 负）、SBC 伪 FCF（FCF−SBC<0）不配用成长估值
- **利润率改善型 / 利润被麻烦压低**（营收增速 << 利润增速，或低谷利润）：用**恢复后正常 margin 的 EPS**，辅以 P/S × 正常 margin、FCF 交叉验，避免拿低谷 EPS 套公式低估
- **高增长封顶**：g≥~22% 一律按 30x，价值框架不追高增长，给不出安全边际就判"买贵了"等回调
- **亏损/周期股**（公式算不了）：用 P/B、净现金/清算价值、中周期正常化盈利打折

参照（合理 PE = min(8.5 + g, 30)，g 用第四步保守前瞻增速）：

| g | 合理 PE | 满仓 ×0.75（伟大） | 满仓 ×0.60（平庸） |
|---|---------|---------|---------|
| 0% | 8.5x | 6.4x | 5.1x |
| 5% | 13.5x | 10.1x | 8.1x |
| 8% | 16.5x | 12.4x | 9.9x |
| 12% | 20.5x | 15.4x | 12.3x |
| 15% | 23.5x | 17.6x | 14.1x |
| ≥21.5% | 30x（封顶） | 22.5x | 18.0x |

## 输出格式

结构化模板 + 表格。**每一步都要深度、用数据说话**，逐表/逐项展开，不许压成"≈Xth 通过"这类一句话概述：
- 一：六指标逐条给实际值 + 阈值 + 是否满足（表格）
- 二：自身 5 年区间表（当前/高/低/中位/前瞻 + 分位）+ 同业对比表（折/溢价%，剔除不可比者）+ de-rating 判断 + 来源 URL
- 三：九宫格 + 利润表逐季 + CF 表 + BS 表，逐表算趋势/比率/点异常季 + GAAP 异常正常化 + 伤口模式
- 四：增长前瞻（历史 CAGR + 季度 YoY + 前瞻共识 + 质量）、护城河（壁垒类型 + 份额趋势 + 威胁）、管理层（任期/增减持/资本配置）、消息面（URL + 日期），熊/牛逻辑写全
- 五：EPS→g→合理 PE(min(8.5+g,30))→合理价→现价→安全边际%→**质量判定（伟大/好公司/平庸 + 理由）**→折扣系数→满仓目标全链条 + 敏感性表 + 护栏检查
- 综合判定：三要素对照表 + **质量判定行**（下）

任何结论后跟数字（增速%、bps、倍数、占比、绝对额），不写"健康/强劲"这类无数字断言。

**综合判定模板**（与开头投资哲学闭环）：

| 要素 | 证据 | 判定 |
|------|------|------|
| 好价格 | 安全边际、正常化 P/E、净现金、分位 | ✓ / ✗ |
| 好公司 | GM 稳定性、FCF、护城河、缩股 | ✓ / ✗ |
| 一次性麻烦 | 暂时（可自愈）还是结构性（持续恶化） | ✓ / ✗ / 存疑 |

表后**必给一行质量判定**（决定折扣系数，summary 与顶部结论都要带）：
> **质量判定**：[伟大 / 好公司 / 平庸]（理由：GM __%、NM __%、FCF yield __%、`FCF−SBC` 正负、护城河__、缩股/稀释__）× 麻烦[明确一次性 / 一般 / 存疑 / 重]　→　折扣系数 **×0.__**　→　满仓目标 $__

归入四类：
- **最佳买点**（三✓）
- **价值陷阱**（仅好价格）：长期不盈利、SBC 伪 FCF、或护城河持续被侵蚀
- **买贵了**（缺好价格）：好公司但正常化估值高于合理价 min(8.5+g,30)，或高增长被 30x 封顶后仍无安全边际
- **结构性恶化**（麻烦非一次性）：GM 崩塌 / FCF 转负 / 护城河永久受损，费雪不合格。看资产托底分两种：
  - 有净现金/清算价值（≈市值）= **困境反转期权**：下行被资产锁死、上行赌反转，盈亏像看涨期权——仅 ≤小仓不对称投机（非价值席位），盯 FCF 能否止血（现金垫会否变薄）
  - 无托底（净负债、有形净资产≈0）= 滑入**价值陷阱**

末尾：是否入池 + 仓位建议 + 建仓/加仓/止损价 + 升级/降级信号。

## 分析报告存档

存至 `financial_data/{STOCK}/analysis.{YYYY-MM-DD}.md`。

**每份报告自包含**：读者只读这一份就能看懂全部结论，正文不得用"见前次报告""同前次"把内容甩给旧报告——熊/牛逻辑、护城河、管理层等都要写全。「与前次变化」表只做 delta 补充，不替代正文。

复分析同一标的：先读最近一份报告，独立完成新分析后在「与前次变化」表标注关键变化。多只对比存 `financial_data/comparison.{YYYY-MM-DD}.md`。

## 汇总所有股票

用户说"汇总所有股票"（或类似：全体股票、汇总一览、给出全体合理价/安全边际）时，读各标的最新 `analysis.{date}.md` + `comparison`，输出下列三块（数据取分析时点，注明会变动）：

**① 主表**（有正常化 EPS 的一般/成长股）：

| 标的 | 现价 | GAAP P/E | 正常化 P/E | g | 合理价 | 安全边际 | 满仓目标（系数） | 结论 |

- **排序：先按档位（可满仓 > 小仓/待催化 > 买贵了），档位内再按安全边际降序。** 不用纯安全边际降序——否则"软/估算的高边际"（如 FCF 口径反推、存疑 ×0.60）会压过"硬/真实的低边际"（如伟大 ×0.75、真实盈利），造成信心错觉
- 安全边际数字硬度不齐时，给软估算标注（如"~+25%·FCF 口径估算"），别让它与硬数字并列误导
- GAAP P/E = 屏幕/报告口径；正常化 P/E = 剔一次性损益/低谷利润后。两者背离要显式呈现
- g = 前瞻可持续增速（不砍半）；合理 PE = min(8.5+g, 30)（可省略此列，g 已隐含）
- 满仓目标 = 合理价 × 质量折扣系数，括注系数（×0.75/0.67/0.60/0.40~0.50）
- 利润分叉/低谷（PDD、BABA 类）给区间（如 $81/$176），不强凑单值
- 多币种（如 0700 腾讯 HKD）标明币种，勿与 USD 标的直接比绝对价

**② 特殊档**（公式不适用：亏损/周期/框架外）：

| 标的 | 现价 | GAAP P/E | 正常化 P/E | 锚 / 买点 | 结论 |

**③ 背离方向**：屏幕 P/E 低估贵（一次性收益垫高利润：投资收益/减值转回 → 正常化后更贵，如 TCOM/MSFT）vs 高估贵（低谷/低基数利润 → 正常化后更便宜，如 BABA/BILI）。基本无背离者一并点明。

## 关键分析偏好

- 货币：非美元财报必须验证单位，报告开头放换算表
- EV/EBITDA：手动核算 EV = 市值 + 债务 − 现金，与 BS 核对
- 股东回报：零回购 + 零分红是重大负面信号（尤其 VIE）
- CEO 公开市场增持 = 最强看多；内部人卖出 = 弱看空（可能税务/遗产规划）
- 仓位模型：10 只 × 10% 等权。核心决策是"是否值得占一个席位"，不搞小仓投机。分批建（先 5% 再 5%），目标仓位就是 10%
