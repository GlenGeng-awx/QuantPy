## 五、安全边际 — 静态锚（合理价 / 满仓目标）

非周期股，新口径 g 不砍半。**正常化 EPS = 正常化 NI ÷ 稀释股数**。TTM 正常化 NI $68.47B（GAAP $90.80B − 税后投资收益 ~$22.3B）÷ 10.85B 股 = **$6.31**。（GAAP EPS $8.36 剔投资收益 → $6.31，「低估贵」背离。）

```
正常化 EPS = $6.31（剔 $28.22B 投资收益，税后）
g（前瞻，不砍半）= 14%
合理 PE   = min(8.5 + 14, 30) = 22.5x
合理价    = 6.31 × 22.5        = $142
质量判定  = 伟大 × 无麻烦（现价高位）→ 折扣系数 ×0.75
满仓目标  = 142 × 0.75          = $107
```

敏感性（g 口径）：

| 前瞻增速 g | 合理 PE | 合理价 | 满仓目标（×0.75） |
|---|---|---|---|
| 12%（历史 CAGR，保守） | 20.5x | $129 | $97 |
| 14%（前瞻共识，基准） | 22.5x | $142 | $107 |
| 16%（AWS 乐观） | 24.5x | $155 | $116 |

交叉验证（利润率扩张型，辅以 P/S）：P/S 3.5x × 正常化 NM ~9-12% → 隐含正常化 P/E ~29-39x，与 38.5x 正常化 P/E 一致 → 确认现价贵。

护栏检查：
- **回购不进 g**：g=14% 用业务增长（AWS + 电商 + 广告）；AMZN 零回购，无缩股干扰 ✓
- **g 质量**：⚠️ **`FCF − SBC = −22.28B < 0`**——严格按护栏「烧钱式增长（FCF 负）不配用成长估值」。但 FCF 负是 capex 主动投入（2027-28 变现），OpInc/OCF 强劲 → 用 OpInc/正常化 EPS 而非 FCF 口径估值，标注 FCF 负为最大保留 ⚠️
- **利润率改善型**：营收 +14% << OpInc/EPS 增速 → 用恢复后正常 margin 的正常化 EPS $6.31，辅以 P/S 交叉验 ✓
- **正常化用真经营 EPS**：剔投资收益 $6.31，不拿 GAAP $8.36 套公式（否则低估贵 → 假便宜） ✓
- **高增长封顶不触发**：g=14% < 21.5%，PE 22.5x ✓
