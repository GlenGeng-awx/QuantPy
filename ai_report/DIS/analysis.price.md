## 一、价格粗筛 — 3/6 过

| 条件 | 阈值 | 实际值 | 满足? |
|------|------|--------|-------|
| 1Y 回撤 | >40% | 22% | ✗ |
| 2Y 回撤 | >60% | 22% | ✗ |
| 距 52 周低点 | ≤15% | +8%（$99.5 vs 低点 $91.7） | ✓ |
| P/E TTM | <15x | 15.9x（E=6.25 GAAP） | ✗（临界，正常化后 15.1x 仍略高） |
| EV/EBITDA | <10x | 11.2x（EV=221.06B，含 $42B 净债） | ✗ |

**3/6，弱入池。** 靠"距 52 周低点 +8%" + "P/S 1.8x" 入池，但回撤浅（仅 22%，远未达深度错杀）+ EV/EBITDA 11.2x（净债推高 EV）说明**这不是深度困境价**——Disney 已从谷底反弹一大截。价格粗筛仅"合格入池观察"，非"深度价值"。

## 五、安全边际（现价代入静态锚）

```
合理价（base）  = $111
现价            = $99.50
安全边际        = 1 − 99.50 / 111 = +10.4%
满仓目标（base）= $74（×0.67 好公司 × 一般偏一次性麻烦）
现价 $99.50 > 满仓目标 $74 → 安全边际不足，未达可满仓线
```

| 价位 | 相对现价 | 含义 |
|---|---|---|
| 满仓目标 $74 | −26% | 需再跌 26% 才到可满仓区 |
| 合理价 $111 | +11% | 上行空间有限 |
| FCF yield @ $99.50 | 7.11B / 172.78B = **4.1%** | 一般（受 parks CapEx 高峰压制；vs ADBE 11.8%）|

**安全边际 +10% 是"软"边际**——建立在 g=8% 与正常化 EPS $6.75 的假设上，且 FCF yield 仅 4.1%（parks CapEx 吞噬）。现价已反映大半困境反转，未提供深度安全垫。

## 综合判定 — 好价格行 + 定档

| 要素 | 证据 | 判定 |
|------|------|------|
| 好价格 | 粗筛 3/6、~0-5th 分位（但含疫情扭曲）、安全边际仅 +10%、FCF yield 4.1%、回撤仅 22% | ✗（不足）|
| （好公司 / 一次性麻烦见 base）| 好公司 ✓（非伟大）；麻烦主体一次性 ✓（夹 linear 结构性） | ✓ / 部分 |

**归类：好公司 + 麻烦主体一次性，但缺好价格 → 待催化/小仓观察。** 现价 $99.50 在满仓目标 $74 与合理价 $111 之间——不是"买贵了"（仍有 +10% 安全边际），但也非"可满仓"（离满仓线 $74 还差 26%）。区别于 ADBE（三✓可满仓）：DIS 质地更杂（GM 37% + linear 拖累）、回撤更浅（22% vs ADBE 47%）、FCF yield 更低（4.1% vs 11.8%）。

**操作**
- **小仓观察区**：现价未达满仓线，先观察或极小仓（<5%）试探
- 加仓价：跌向满仓目标 $74（−26%）分批建；$85 以下（乐观情形满仓线）可先小仓
- 止损/降级：见 base 降级信号（streaming OI 转负 / parks OpInc 连续负 / 净债威胁评级）
- 升级：见 base（streaming OpM 破 10% 续升 + parks 客流回正 + 调整 EPS 续双位数）→ 上调 g 至 10%、合理价 $128、满仓 $86
