## 二、估值分位（区间结构 + 同业 + de-rating）

**非周期股**（媒体娱乐消费，parks 有轻周期性但非商品周期）→ 走正常分位。当前 P/E 与所处分位见 price 文件。

当前 GAAP P/E ~15.9x 处 5 年 **~0-5th 分位**（低于 2022-2025 每一年）。10 年均值 45x 被疫情泡沫严重扭曲，参考价值低。

⚠️ **de-rating 陷阱**：2021-2023 的高 P/E 全部来自"疫情利润被砸到地板 → 分母极小"的机械放大（周期性利润坍塌型高 P/E），**不是成长溢价**。因此"从 266x 跌到 16x"绝大部分是利润恢复的正常回归，而非市场杀估值。真实可比的是利润正常化后的 2025 (17.8x) 与当前 (15.9x)——16x 对一家 parks 皇冠 + streaming 刚转正 + 双位数调整 EPS 增长的公司是合理偏低，但**非 ADBE 那种绝对深度低位**（Disney GM 仅 37%、linear 结构衰退，不配 89% GM 软件的估值）。

- **GM 稳定 35.8-38.6%**——媒体娱乐典型（内容成本 + parks 运营成本重），远低于软件，但无恶化
- **营收稳步 +6.5% YoY**（Q1'26 $25.17B）——parks + streaming 双引擎推动，未见崩塌
- **⚠️ Q2'25 税收一次性 −$2.73B provision**（递延税项转回）把当季 NI 拉到 $5.26B、EPS $2.92——这是 TTM EPS YoY 显示 −8.8% 的**真正原因**：不是经营恶化，而是去年同期基数被一次性税收利得垫高。剔除后底层经营 OpInc 稳步增长（Q1'26 OpInc +8% YoY）
- **缩股刚起步**：dil shares 1.81B->1.77B（5Q −2.2%），FY23-24 零回购，FY25 重启（$7B 授权）
- 正常化：TTM Unusual −$806M（重组/减值残余），Normalized Income $11.71B → 正常化 EPS ≈ $6.54；叠加剔除 FY25 税收异常，**正常化 EPS 锚 $6.75**（对齐 mgmt FY26 调整 EPS 指引 ~$6.91 的保守取值）

### 分部拆解（FY26 Q2，thesis 核心）

| 分部 | 营收 | OpInc | YoY | 解读 |
|------|------|-------|-----|------|
| Experiences（parks/游轮/消费品）| $9.5B | $2.6B | OI +5% | **皇冠 jewel，高利润护城河**；国内客流 −1%（国际访客软），Q3 指引改善 |
| DTC（Disney+/Hulu）| $5.49B | $582M | **OI +88%** | **盈利拐点确认**，涨价驱动，FY26 目标 OpM >=10% |
| Linear TV | 衰退 | 下滑 | 结构性 | 剪线持续，结构性衰退（非一次性）|

**核心 thesis**：streaming 从烧钱转为 OI +88% 加速盈利（一次性拐点，利好），parks 韧性高利润（护城河），linear 结构性衰退（永久逆风）。三力交织 → 混合体，非纯困境反转、非纯成长。

### 伤口模式

Income 3yr 满分（利润从疫情谷底翻 5x）+ TTM/5Q 因一次性税收基数扣分 = **困境反转已大半完成**。streaming 转正是拐点证据。linear 衰退是结构性伤口（永久），但被 parks + streaming 稀释。整体：**情绪+周期伤口大部分已愈，残留 linear 结构性逆风**。

### 护城河 — parks 极宽 / 内容 IP 无可复制 / 整体因 linear 稀释

- **壁垒类型**：
  - **Parks（皇冠）**：不可复制的实体资产 + IP 体验 + 定价权（几十年提价史），护城河极宽，Universal 是唯一有力竞争但规模远小
  - **内容 IP 库**：Marvel/Star Wars/Pixar/Disney 动画 + Fox 库，无可复制，跨 parks/streaming/消费品变现飞轮
  - **Streaming**：Disney+/Hulu/ESPN Flagship，规模第二梯队（NFLX 之后），但 IP 独家内容护城河
- **份额趋势**：parks 稳固领先；streaming 追赶中已转正；linear 份额随行业剪线永久流失
- **威胁**：linear TV 结构性衰退（永久）；streaming 竞争激烈（NFLX/Amazon/Apple）；parks 国际客流软 + Universal Epic Universe 分流

## 五、安全边际 — 静态锚（合理价 / 满仓目标）

非周期股，新口径 g 不砍半。正常化 EPS = \$6.75（TTM 正常化 \$6.54 剔 FY25 税收异常，对齐 mgmt FY26 调整 EPS 指引 ~\$6.91 的保守取值）。

```
正常化 EPS = $6.75
g（前瞻,不砍半）= 8%              # 营收 ~5% + streaming margin 一次性台阶,剔回购驱动 EPS
合理 PE   = min(8.5 + 8, 30) = 16.5x
合理价    = 6.75 x 16.5          = $111
质量判定  = 好公司 x 一般偏一次性  -> 折扣系数 x0.67
满仓目标  = 111 x 0.67           = $74
```

敏感性（g 口径 / EPS 口径）：

| 情形 | 正常化 EPS | g | 合理 PE | 合理价 | 满仓目标（×0.67）|
|---|---|---|---|---|---|
| 保守（基准）| \$6.75 | 8% | 16.5x | \$111 | \$74 |

| 悲观（纯 TTM 正常化 + 低 g）| \$6.54 | 6% | 14.5x | \$95 | \$64 |

护栏检查：
- **回购不进 g**：g=8% 用业务 + streaming margin，缩股价值体现在安全边际 + EPS 基数，不抬合理 PE ✓
- **g 质量**：`FCF − SBC = +\$5.66B > 0` 且 3 年增长 ✓；但 FCF 受 parks CapEx 高峰压制，PEG ~2.0（不便宜）

- **无净现金托底**：净债 \$42B，估值靠盈利/FCF 非资产，杠杆真实但利息覆盖 7.9x 安全 ✓
- **linear 结构性衰退**：营收 g 已反映（~5%），不给成长溢价 ✓
