**质量判定**：**周期股（费雪✗）**——GM 从 FY23 −9.1% 摆到 TTM 84.6%，**无定价权、纯随内存价周期**；三寡头（三星/SK 海力士/美光）但产品同质、资本开支黑洞（TTM CapEx $25B）。**不适用伟大/好公司/平庸 × 折扣系数口径**，改用**中周期正常化盈利 + P/B 分位 + 净现金**锚。麻烦不存在——是超级顺境 euphoria，与"困境反转"完全相反。

**核心锚（静态）**：中周期正常化 NI **$12-20B** → 中周期 EPS **$10.5-17.5**（1.14B 股）→ 中周期合理 PE **12-13x**（内存历史极少 >15x）→ **中周期合理价 $126-228**；P/B 峰值 ~3x 对应 ~$265、P/B 底部 ~1x 对应 ~$88。**周期顶部，回避（框架外）。**

## 一、周期定位判定 — 极端内存超级周期顶部（先定性）

**判定周期股**：内存（DRAM/NAND/HBM）盈利随存储器现货价剧烈摆动，是典型周期股。当前处**超级周期绝对顶部区**：AI/HBM 驱动的内存荒把价格推到历史极值，下季指引营收再创纪录。一年从 52 周低点 $103.38 涨到近高点区（+800%+），是本组合里唯一"暴涨到顶"的标的，与"因困境下跌"的猎场定义完全相反。

## 二、估值分位（周期股：中周期正常化 + P/B 分位 + 净现金，P/E 不作贵贱信号）

**P/E 会误导**——底部利润趋零/亏损→P/E 高或负（买点），顶部利润暴涨→P/E 低（卖点）。**当前 P/E 22x 看似不贵，恰恰是超级周期峰值盈利把分母吹大造成的"顶部低 P/E"假象**。周期股顶部特征：P/E 分位低 + P/B 分位极高。

### 跨周期盈利中枢（看摆动幅度）

| 财年 | NI | EPS | 周期位置 |
|------|-----|-----|---------|
| FY2022 | 8.69B | 7.75 | 上一轮高峰 |
| FY2023 | **−5.83B** | −5.34 | **深谷（亏损）** |
| FY2024 | 0.78B | 0.70 | 复苏起点 |
| FY2025 | 8.54B | 7.59 | 上行 |
| **TTM FY2026** | **50.47B** | **44.24** | **超级尖峰**（AI 内存荒） |

- FY23→TTM，NI 从 −5.83B 到 +50.47B——**这就是周期股，没有"稳定盈利"可套 PE**
- 季度轨迹更触目：营收 9.30→11.31→13.64→23.86→**41.46B**，COGS 基本不动（~$6B）→ GM 37.8%→44.7%→56%→74.4%→**84.6%**。**价格暴涨全额落入毛利**（内存超级周期典型形态：产能固定、价格数倍、成本不变 → 利润爆炸）

### P/B 分位 — 极端高位（周期顶部铁证）

- P/B = 现价 $975.56 ÷ bookValue $64.24 = **~15.2x**（MCap $1,101.79B ÷ 权益 $100.72B ≈ 10.9x，取报告 P/B 15.2x）
- 美光历史 P/B 区间约 1-3x（底部 ~1x、常态峰值 ~2-3x）。**15x 完全脱离历史区间**，因股价已 price in 超级周期永续，而账面权益（$100.7B）尚未追上盈利爆炸
- 周期股口径：**P/E 分位低 + P/B 分位极高 = 周期顶部**（与"P/E>70% + P/B<30% = 底部"完全相反）

### 净现金

- 现金 $26.02B − 总债 $6.38B = 净现金 ~$19.6B，仅占 MCap **~1.8%** → 无托底意义（周期底部才会出现净现金/市值 >30% 的买点）

## 四、周期定位 / 护城河 / 管理层 / 消息面

### 周期定位（替代"增长前瞻"）

- **当前位置：超级周期绝对顶部区**。AI/HBM 驱动的内存荒把价格推到历史极值，下季指引再创纪录
- **前瞻风险**：超级周期见顶后回落往往剧烈。触发点——新产能上线（美光/三星/SK 海力士 2026-2027 大扩 HBM/DRAM）、AI capex 若进消化期、通用 DRAM/NAND 需求周期性回落
- **无"可持续增速 g"可用**：周期股不适用成长估值公式，increments 是价格周期不是结构增长

### 护城河 — 弱（大宗商品 + 寡头）

- 三寡头（三星、SK 海力士、美光）控 DRAM ~95%，理论上有寡头定价纪律，但历史反复证明**产能竞赛会打破纪律**（GM 摆到负值）
- HBM 是相对差异化的亮点（AI 加速器绑定、认证壁垒），比通用 DRAM 粘性高——**这是本轮可能"结构性抬高盈利底"的唯一论据**，但仍是产能可扩、多供应商竞争的产品
- 费雪口径：GM 从 −9% 到 +84% 的剧烈摆动 = **无定价权 = 非好公司**（顺周期的伟大是幻觉）

### 管理层

| 项目 | 事实 |
|------|------|
| 资本配置 | 用尖峰现金去杠杆（债 15.28→6.38B）✓明智；回购克制（不高位接盘）✓；CapEx $25B 扩产（行业性，双刃剑） |
| SBC | 1.3%/Rev，低 |
| 周期择时 | 管理层不大回购是好信号——暗示自己也知道是高位 |

### 消息面

- 2026-06 FY26 Q3 财报爆表（营收 $41.46B 翻两番、EPS $24.67 远超预期），股价大涨
- 内存超级周期主线：AI 服务器 HBM 需求 + 通用内存荒共振
- 全行业扩产进行中——供给回归是中期最大风险

### 熊 / 牛逻辑

**熊市逻辑（周期顶部）**：GM 84.6%/NM 68% 是物理上不可持续的尖峰；P/B 15x 脱离历史区间 5x+；一年 +800%+ 是 euphoria；全行业疯狂扩产 → 供给过剩必然 → 下一轮价格崩 + 亏损（FY23 就是 −5.83B）；中周期正常化合理价 $126-228，现价是其 4-8 倍；周期股在"财报最靓 + P/E 最低"时见顶，正是此刻。

**牛市逻辑（超级周期永续论）**：AI/HBM 需求结构性抬高内存 earnings floor，"这次不一样"；HBM 差异化 + 认证壁垒使美光议价力提升；三寡头这轮更有纪律、扩产克制；净现金 + 去杠杆使资产表最强。**但——即便牛论全对，也需内存价永久维持历史极值才撑得起近 $1,000 现价；任何均值回归都是腰斩再腰斩。周期股赌"永续尖峰"是最经典的顶部错误。**

## 五、安全边际 — 静态锚（周期股：中周期正常化 + P/B，非 min(8.5+g,30) 公式）

**周期股不套 min(8.5+g,30) 公式**（无稳定 g、无正常化 EPS）。用中周期锚 + P/B 回归：

| 锚 | 数值 | 相对现价（见 price） |
|---|---|---|
| 中周期正常化合理价（NI $12-20B × 12-13x） | $126-228 | 高估数倍 |
| P/B 回归（历史峰值 P/B ~3x × 权益 $100.7B ÷ 1.14B） | ~$265 | 高估 |
| P/B 底部（~1x） | ~$88 | 高估 |
| 52 周低点（上一轮周期起点） | $103.38 | — |

护栏检查：
- **周期股口径**：P/E 反读——22x 低 P/E = 峰值盈利 = 卖出信号，非便宜 ✓
- **底部 ≠ 任意价格买；顶部更是绝不追**：CLAUDE.md 明确周期股须在中周期口径有安全边际，当前是负安全边际的极端顶部
- **净现金锚**：$19.6B 仅 ~1.8% MCap，无托底（周期底部才会出现净现金/市值 >30% 的买点）
- **无可持续 g**：increments 是价格周期，不配成长估值
- **中周期正常化 NI $12-20B 已慷慨**（含 HBM 结构性抬升）——即便把 TTM $50B 尖峰当新常态（几乎不可能，全行业疯狂扩产），也只是"顶部合理"，而峰值盈利不可持续 = 顶部合理 P/E 是卖出信号
