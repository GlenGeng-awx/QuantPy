### 伤口模式

3yr Income 满分（真实盈利力）+ TTM/5Q 崩（Cidara 会计冲减）= **表面像"伤口反转候选"，实为一次性会计噪音**。但真正的伤口在未来：**Keytruda 2028 悬崖是结构性的、尚未反映在报表里**。这不是"3yr 高 + TTM 低"的困境反转，而是"当下好 + 未来结构性下滑"——方向相反。

## 五、安全边际 — 静态锚（合理价 / 满仓目标）

**关键：g 不能用历史，必须用悬崖压制的前瞻。** 正常化 EPS 用 FY25 GAAP $7.28（Q1'26 $9B Cidara 一次性剔除）。

```

合理 PE   = min(8.5 + 1.5, 30) = 10.0x
合理价    = 7.28 × 10.0        = $73
```
