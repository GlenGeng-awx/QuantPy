## 五、安全边际 — 框架外（净现金地板 + 管线期权，非 EPS 公式）

**公式不适用**（亏损、营收 lumpy、无正常化 EPS）。用净现金地板 + 管线期权定价：

```
净现金地板  = €14.34B × 1.143 / 253M = $64.7/股   （现金+短投，留存收益来源，算数不打折）
含长期证投  = ~€16B × 1.143 / 253M   = ~$72/股     （管理层口径含长期证券）
现价        = $97.55
管线期权定价 = 97.55 − 64.7 = $32.9/股（占现价 34%）
```

**不对称结构（看涨期权型）**：
- **下行**：跌向净现金 $64.7（−34%），再往下需 believe 现金被烧光（BMS cost-sharing 下短期难）→ **下行被现金锁死**
- **上行**：BNT327 及管线兑现 → 远超现价（IO backbone 潜力）
- **中性**：管线平庸、现金慢烧 → 股价在 $65-100 区间磨

**出手纪律（非价值席位，仅困境反转期权）**：
- 现价 $97.55 含 34% 管线期权溢价——**非纯现金地板价买入**，为管线付了 $32.9/股
- 理想不对称买点：**接近净现金 $65-72（期权几乎白送）** 才是真安全边际
- 现价 $97.55 需 believe 管线期权 $32.9/股 值这个价——**属投机，非价值**

护栏检查：
- **亏损/框架外**：不套 EPS 公式，用净现金 + 期权 ✓
- **现金来源**：COVID 留存收益（最优质），作锚算数不打折 ✓
- **烧钱风险**：R&D €2.6-2.8B/年，但 BMS cost-sharing + $2B 分期显著延长跑道，现金垫短期不会快速变薄 ✓
- **无浮存虚增**：现金是真金（疫苗销售留存），非应付膨胀 ✓
- **仓位**：CLAUDE.md——困境反转期权仅 ≤小仓不对称投机（非 10% 价值席位） ✓
