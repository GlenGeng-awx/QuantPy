## 关键背景（决定全篇基调）

INTC 2025 曾跌至 **~$19**（谷底），此后在 Lip-Bu Tan 换帅 + **英伟达 $5B 增持 + 软银 $2B（$23/股）+ 美国政府 9.9% 股权（$5.7B）** 三重背书下暴涨，YTD 2026 +278%、过去一年 **+456%**，现价 ~$120，距 52 周高 $142 仅 **15%**。**这不是"跌下来的好公司"，是"已 6 倍反弹、把反转预期全部计入的困境股"。** 框架猎场是"因暂时困境**下跌**的好公司"——INTC 恰好相反：先崩后暴涨。所以第一步就大概率不入池，第五步无安全边际。

- **GM 从历史 60%+ 崩到 27-39%**——制程落后 + 代工低毛利拖累，定价权丧失（vs ADBE 89% 纹丝不动）。这是**结构性伤口**（真实竞争侵蚀），非情绪伤口

## 五、安全边际 — 静态锚（公式不适用）

**亏损 + 未证实反转 + 周期股，成长公式给不出正常化 EPS**（TTM 亏损，Normalized 连续为负）。用资产 + 中周期正常化盈利打折：

```
公式法    = 不适用（TTM EPS 负，Normalized 连续为负；2026E $1.09 → 前瞻 P/E 110x，无意义）
P/B 锚    = 现价 P/B 2.47x、P/有形账面 3.75x（恰在 10 年中位）→ 不便宜，已隐含反转
中周期锚  = 若假设反转成功、恢复到"正常半导体" ~15% NM × $55B 营收 ≈ $8B NI ÷ 5B 股 ≈ EPS $1.6
           给成熟半导体 15-20x → $24-32（乐观情景仍远低于现价 $120）
资产托底  = 有形账面被巨额可能减值 fab 占据 + 现金靠发债增发（最差来源）→ 托底不牢
```

**结论：现价 $120 已充分甚至过度计入反转成功。** 反转失败（14A 无客户）→ 回到 P/B <1x（$40-50 区）；反转成功 → 乐观中周期 $24-32 EPS-based 也需多年兑现。**下行远大于安全边际，非价值买点。**
