### 护城河 — 中等偏宽（HIV 特许经营）

- 壁垒类型：HIV **Biktarvy 美国份额 >52%**（每季 YoY 增长，自上市以来）、专利 + 品牌 + 医生粘性、lenacapavir（Sunlenca/Yeztugo）长效给药（每半年一次 PrEP，首创）技术壁垒
- 份额趋势：Biktarvy 稳居 HIV 首位；**Biktarvy 专利经和解延至 2036**（原 2033 悬崖，与 Lupin/Cipla/Laurus 和解推迟仿制药至 2036-04）-> 现金牛跑道延长
- 威胁：⚠️ **HIV 化合物终局专利悬崖**（Biktarvy 2036 后）；Trodelvy 肿瘤扩张受挫（2024 撤回膀胱癌）；ViiV（GSK）HIV 竞争（$1.25B 专利和解）；lenacapavir 需成功接棒 Biktarvy 才能延续护城河

护城河**真实但非无限**——HIV 是"高粘性成熟特许经营"，靠 lenacapavir 换代续命。不是 GLP-1 那种扩张市场，是稳态守成。

## 五、安全边际 — 静态锚（合理价 / 满仓目标）

一般股（有正常化 EPS）。g 不砍半。剔 2024 一次性 IPR&D 减值。

```
正常化 EPS = $7.73         # TTM 正常化 NI $9.66B / 1.25B 股（剔 CymaBay 减值等）
g（前瞻,不砍半）= 5%        # 第四步产出:营收 ~3-5%,HIV 8% + Yeztugo 增量,成熟基数拖累
合理 PE   = min(8.5 + 5, 30) = 13.5x
合理价    = 7.73 x 13.5      = $104
质量判定  = 好公司 x 无明显麻烦 -> 折扣系数 x0.67
满仓目标  = 104 x 0.67       = $70
```

敏感性（g 口径）：

| 前瞻 g | 合理 PE | 合理价 | 满仓目标(x0.67) | vs 现价 $131 |
|---|---|---|---|---|
| 4%（保守，纯 HIV 成熟） | 12.5x | $97 | $65 | 现价高 35% |
| **5%（基准）** | **13.5x** | **$104** | **$70** | 现价高 26% |
| 8%（乐观，Yeztugo 大放量） | 16.5x | $128 | $85 | 现价仍高 2% |

**即便最乐观 g=8%，合理价 $128 仍 < 现价 $131** -> 无论如何算，GILD 无安全边际。

护栏检查：
- **回购不进 g**：g=5% 用业务增长，GILD 回购温和（~1%）+ 分红为主 ✓
- **g 质量**：`FCF - SBC = +$9.33B > 0` ✓，但增速低（成长估值不加分，靠现金牛+分红）
- **正常化避免减值扭曲**：用 normEPS $7.73（非 2024 GAAP $0.38 或 TTM GAAP $7.35），避免 2024 IPR&D 减值扭曲 ✓
- **无净现金托底**：净负债 $14.5B，但现金牛+利息覆盖 11.9x 可控；分红有保障 ✓
