# GS（高盛 Goldman Sachs）— 基本面 base

> **静态层**：财报/前瞻/护城河/管理层驱动，季度或重大事件更新。动态见 `analysis.price.md`。
> **更新: 2026-07-04**（财报 FY26 Q1，2026-04-13 发）　货币: USD　**金融股口径 + 强周期投行**（P/B 替代 EV/EBITDA；FCF/现金流公式不适用；盈利随资本市场周期剧烈摆动）

**质量判定**：**好公司（顶级投行，但强周期 + 高杠杆 + 盈利波动大）**——全球投行/交易/M&A 顾问王者、Q1'26 record equities trading + IB 费 +48%、ROE 14.6%（Q1 annualized 19.8%，周期高位）；但 GS 是**周期股中的周期股**——IB/交易营收随资本市场牛熊剧烈摆动（2023 ROE 仅 7.4%、NI $8.5B vs 2026 TTM $18B），非稳定复利机器。用 **正常化（中周期）P/E + P/B + ROE** 锚，**P/E 反读**（当前 record M&A 周期 = 峰值盈利，18.7x 是"顶部偏高倍数"）。银行无干净 FCF（OCF −40B 是交易资产流动）。× 麻烦 **无**（近 52 周高位，1Y 回撤仅 9%，record 盈利周期顶）

**核心锚（静态）**：正常化（中周期）EPS ~**$48** → 投行合理 PE **~11x**（周期波动大给低倍）→ **合理价 ~$530**；P/B 交叉验 合理 ~1.4x × 账面 $370/股 = **$520**。价值买点 P/B ~1.0x ≈ **$370**（账面附近，投行历史危机买点）

## 二、估值 — 金融股用 P/B + P/E 反读（周期顶）

| 口径 | 数值 | 说明 |
|------|------|------|
| P/E TTM | 18.7x（P=$1021 / E=$54.75） | 投行历史 8-13x，18.7x **明显偏高**——但周期股 P/E 反读：record 盈利下高 P/E ≠ 贵，需看是否周期顶 |
| **P/B** | **~2.48x**（$1021 / 账面 $370.29） | 投行核心；**10 年 TBV 中位 1.19x**，当前 **149% 上方**——极端高位 |
| P/TBV | ~2.92x | 145% above 10 年中位 1.19x（[GuruFocus](https://www.gurufocus.com/term/price-to-tangible-book/GS)） |
| ROE | 14.6%（TTM）/ Q1 annualized 19.8% | 周期高位——2023 谷底仅 7.4% |

金融股 + 强周期口径：**P/B 2.48x 是关键警报**——GS 传统在 1.0-1.3x P/B（投行盈利波动大，市场给低倍），当前 2.48x 处历史极端高位（149% above 10 年中位）。这不是"P/E 反读的便宜周期底"，而是**盈利在周期顶（record M&A + trading）+ 估值也在顶（P/B 2.48x）双重高位**——最危险的组合。

**自身 P/B 5 年区间**：约 0.9x（2020/2022 低点）~ 2.5x（2026 record 高位），10 年中位 1.19x（[MacroTrends](https://www.macrotrends.net/stocks/charts/GS/goldman-sachs/price-book)）。当前 2.48x 处 5 年 **~95th+ 分位**（极端高位）。

**同业对比（P/B / ROE）**：

| 投行/银行 | P/B | ROE | 备注 |
|------|-----|-----|------|
| **GS** | **~2.48x** | 14.6%（周期顶） | P/B 处历史极端高位 |
| MS（摩根士丹利） | ~2.2x | ~15% | 财富管理占比高更稳，P/B 相近 |
| JPM | ~2.46x | 16.2% | 更稳更高 ROE，同样 P/B 更合理 |

GS 的 2.48x P/B 与 JPM 相当，但 GS 盈利周期性远高于 JPM（IB/trading vs 多元化 NII）——同样 P/B 下 GS 风险更大。

⚠️ **de-rating 陷阱反向**：GS 历史 P/B 中位仅 1.19x，当前 2.48x 是"AI/IPO/M&A 复兴叙事 + Basel III 松绑释放资本"推高的**周期+叙事双溢价**，一旦资本市场转冷，盈利 + 估值双杀（2022 曾从峰值腰斩）。

## 五、安全边际 — 投行锚（中周期正常化 P/E + P/B），公式不适用

```
min(8.5+g, 30) 公式对投行不适用（无干净 FCF；盈利强周期）。用中周期锚：
正常化（中周期）EPS ≈ $48（TTM $54.75 / normalized $58.9 反向下调——剔 record M&A/trading 周期顶，取跨周期中枢）
投行合理 PE ≈ 11x（盈利波动大，市场给低倍；GS 历史 8-13x）
合理价 ≈ 48 × 11 = ~$530
P/B 交叉验：合理 P/B ~1.4x（略高于 10 年中位 1.19，给品牌溢价）× 账面 $370/股 = $518 ✓（两法吻合 ~$520-530）
价值买点：P/B ~1.0x ≈ $370（账面附近，投行历史危机买点；2020/2022 曾触及）
```

护栏：
- **金融股 P/B 口径**：现价 P/B 2.48x 处 95th+ 分位、10 年中位 149% 上方 = 严重买贵；买点在 P/B ~1.0x ✓
- **P/E 反读 + 周期顶**：18.7x 是 record 盈利下的高倍数——**周期顶不是"P/E 高=便宜"的底部**，是盈利+估值双顶，正常化后更贵 ✓
- **周期正常化反向下调**：record M&A/trading 不可持续，正常化 EPS 从 $58.9 下调至 $48（区别于 BAC 温和下调） ✓
- **超高杠杆尾部风险**：D/E 3.52，投行 2008 式尾部风险 ✓
- **无 FCF 锚**：投行现金流表不可用，勿套 FCF yield ✓
