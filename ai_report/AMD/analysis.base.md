## 一、周期判断（半导体）

AMD 是半导体，**强周期**。但当前**不在中周期中枢，也不在顶/底的极端**——处于 **AI 数据中心结构性上行的早-中段**：
- 数据中心营收 2025 ~$16B → 2026E ~$22.9B → 2027E ~$33.9B，公司指引数据中心 CAGR >60%、AI 营收 CAGR >80%（[Seeking Alpha 2026](https://seekingalpha.com/news/4546760-amd-outlines-path-to-60-percent-annual-data-center-growth-and-multi-billion-ai-revenue-by)）
- 与 MU（内存周期**顶部**、P/B 15x 极端）不同，AMD 是**逻辑芯片 + AI 加速器结构性成长**，利润非"周期顶低质量利润"。但估值已把多年高增长 price-in
- 结论：不是周期底买点，也非周期顶卖点；是**成长顶部估值**——用成长框架（g 封顶 30x），非中周期正常化 P/E

AMD 无 de-rating——反而是 **re-rating 到历史极高**。市场对 AI 数据中心 CAGR >60% + MI400/MI450 系列（2026 Q3 起 Helios 机架级）给了满分成长溢价。任何 AI 资本开支放缓或 MI 系列执行不及预期 → 61x 将剧烈收缩。这是"高增长小盈利→范式切换"的**上行泡沫端**，非低估。

- Xilinx 收购无形摊销 **~$1B/季 ≈ $4B/年**（税前）跑进 GAAP，是 GAAP↔非 GAAP 差异主因

- **背离方向**：GAAP P/E 170x「看起来极贵」，正常化后 ~101x「仍极贵」——**正常化后依旧买贵了**，只是没有 170x 那么夸张

### 护城河 — 中等（老二困境）

- 壁垒类型：x86 CPU 双寡头（与 Intel）+ GPU/AI 加速器（挑战 NVDA CUDA 生态）+ Xilinx FPGA/自适应计算
- 份额趋势：服务器 CPU 份额持续从 Intel 抢（EPYC 强）；**AI GPU 份额 <20%**，NVDA CUDA 软件生态 + 网络（NVLink）是 AMD 最大短板（ROCm 生态弱于 CUDA）
- 威胁：NVDA 生态锁定极强；超大规模厂商自研 ASIC（含 AVGO/Marvell 定制）；Intel 代工/产品回血
- 定性：**有护城河但非宽**——AI 加速器是"挑战者"不是"垄断者"，与 ASML 的 EUV 独家垄断天壤之别

## 五、安全边际 — 静态锚（合理价 / 满仓目标）

成长股（AI 半导体），g 不砍半。用**正常化（非 GAAP）EPS**（GAAP 被 Xilinx 摊销压低失真）。

```
正常化 EPS = GAAP NI $5.01B + 摊销税后加回 ~$3.4B = $8.4B / 1.64B 股 = $5.13
g（前瞻，不砍半）= >35%（公司指引），远超 22% 上限
合理 PE   = min(8.5 + 35, 30) = 30x（价值框架高增长硬封顶）
合理价    = 5.13 × 30           = $154
质量判定  = 好公司 × 无麻烦(反而追高)  -> 折扣系数 ×0.67
满仓目标  = 154 × 0.67          = $103
```

敏感性（口径/EPS）：

| EPS 口径 | 值 | 合理价（30x 封顶） | 满仓（×0.67） | 现价 $519.50 安全边际 |
|---|---|---|---|---|
| 正常化 TTM（非 GAAP） | $5.13 | $154 | $103 | **−238%** |
| Forward 2026E 非 GAAP | ~$6.50 | $195 | $131 | **−166%** |

护栏检查：
- **回购不进 g**：g 用业务增长（数据中心 >60%），回购未真缩股，不适用
- **g 质量**：`FCF − SBC = +$6.8B > 0` 且 3 年高增长，但成长已被市场充分定价
- **高增长封顶**：g >35% ≥22% -> 一律 30x，30x 后仍无安全边际 -> **判"买贵了"等回调**
- **周期护栏**：AI 数据中心结构性成长，非周期顶低质量利润；但估值把多年成长 price-in，需 >$20 EPS 兑现才支撑现价
- **背离护栏**：GAAP P/E 170x 失真（Xilinx 摊销），正常化 101x 才是真实——即便正常化仍极贵
