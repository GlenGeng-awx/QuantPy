### SCORECARD 背离检查（机械信号 → 原始财报证实/推翻 → 机会/陷阱）

PDD 是本次两只里**背离最丰富**的：屏幕便宜（P/E 8.7x、净现金 56%）诱人，但逐维度按图索骥后是"机会与陷阱并存"，不能只看便宜。

| 维度 | SCORECARD 说（机械） | 原始财报查（证实/推翻） | 方向 → 结论 |
|------|--------------------|----------------------|-------------|
| **Income 3yr GM Trend ✗**（63%→60.9%→56.3%） | 唯一被机械打到 ✗ 的项：毛利率逐年掉 | ✅ **证实且更严重**：拉长看 **75.9%(2022)→56.0%(TTM)**，是 Temu（低毛利、物流重）占比升 + 免运费 + ¥100B 供应链投入的**模式切换结构性压缩**，非周期波动。机械 ✗ 方向对，但"机械没充分反映其结构性"——评分把它当普通下滑扣分，实则是护城河质地的永久性稀释 | **机械丑 + 实际更丑（结构性）= 同向且机械低估严重性**。**别只看便宜**：这是最大真实风险，压住"好公司"评级到 △ |

**净结论**：PDD 的背离是双向并存——① **GM ✗ 机械低估了结构性严重程度**（别只看 8x 便宜，护城河在被 Temu 稀释）；② **EPS/NM 屏幕断崖是非经营噪音+投入台阶**（局部错杀，正常化后经营其实在长）；③ **BS/CF 满分掩盖零回购**（现金真但不回流 = 价值实现陷阱）。综合 = **深价值现金牛，但 GM 结构下滑 + 零回购把"好公司"钉死在 △**，不是干净的 ADBE 式错杀。**便宜是真的，机械没充分反映的两个风险（GM 结构性 + 现金不可及）也是真的**——故 ×0.60 平庸档、小仓非满仓。

## 五、安全边际 — 静态锚（高度 g 敏感 + 净现金底 + EV/盈利交叉验）

非周期股，min(8.5+g,30) 适用。全程股价（ADS）口径。

```
股数      = ~1.42B ADS
正常化 NI = 97.18B CNY ÷ 7.1 = $13.69B
正常化 EPS = 13.69B ÷ 1.42B = $9.50 / ADS（与 combine E=$9.49 一致）
正常化 P/E = 8.7x（屏幕）

g（前瞻，营收减速 ~10% + GM 压缩拖累）= 5%（基准）
合理 PE = min(8.5 + 2×5, 30) = 18.5x       # 注：本标的沿用旧敏感表 8.5+2g 展示 g 弹性
合理价 = 9.50 × 18.5 = $176
质量判定 = 平庸（现金牛带瑕疵）× 半结构 + 无催化 → 折扣系数 ×0.60
满仓目标 = 176 × 0.60 = $105（g5% 口径）
```

敏感性（PDD 全看 g——投入是否毁盈利能力是焦点）：

|---|---|---|---|---|

**净现金底 + EV 交叉验（更硬的下行锚）**：

| 锚 | 值 | 判定 |
|----|-----|------|
| 净现金/ADS | $60.7B ÷ 1.42B = **~$42.7/ADS** | 现价中约一半是净现金 |

- **即便最悲观（g=0，投入永久压毛利），合理价 $81 仍略高于现价区间**，且净现金 ~$42.7/ADS 是独立硬底——下行被锁。
- **EV/正常化盈利 ~3.5x** 是全标的最便宜的经营业务定价：市场几乎只按现金 + 一点点给国内现金牛估值，Temu 当负期权算。
- 基准 g=5% 满仓目标 $105——但 **g 兑现取决于 ¥100B 投入是否转化 + Temu 是否止血**，故口径分歧极大（$81~$271）。
- **唯一压住乐观的是"零回购无催化"**：便宜可能长期不兑现（PDD 与 ADBE/PYPL 真缩股的本质差距）。
