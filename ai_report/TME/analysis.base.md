- **GM 持续扩张：35.3%（2023）→42.3%→44.2%→44.4%（TTM）**——高毛利音乐订阅替代低毛利社交娱乐/直播分成，模式优化的结构性改善。
- **OpInc 3 年 CAGR +31%**（3.53→4.97→7.44→9.74B），OpM 稳定 ~30%——经营引擎强劲。
- **★正常化陷阱（核心认知）**：Q1'25 含 ¥2.46B 一次性投资收益（Other Income 暴增），使 FY25 报告 NI 11.06B 虚高（正常化 8.91B，unusual 2.51B）。**TTM（2026-03 = Q2'25~Q1'26）这 4 个季度是干净的**（Other Income 合计仅 +204M），但同比的"去年"含 Q1'25 那笔 → 造成 **TTM EBITDA −24%、EPS −20% 的假象**。剔除基数效应，经营实际在增长（OpInc TTM +2.4%、Q1'26 +11.2%）。
- **正常化 NI ≈ 8.86B**（TTM 干净季度之和；combine 的"Normalized 6.79B"是误把 FY25 调整套到 TTM，实际那笔已滚出窗口）。

### EV/EBITDA 手算（纠正 combine glitch）

combine 显示 −0.5（D&A 缺 + 货币混算 glitch）。手算：净现金 21.90B CNY（Cash+STI 26.74B − 债 4.84B）= $3.08B；EV = MCap 14.13B − 3.08B = **$11.05B**；EBITDA TTM 11.18B CNY = $1.575B → **EV/EBITDA ~7.0x**（<10x，✓）。剥离现金后经营业务仍便宜。

## 五、安全边际 — 静态锚（合理价 / 满仓目标）

非周期股，新口径 g 不砍半。全程股价（ADS）口径。

```
股数       = MCap ÷ 现价（时点） ≈ 1.63B ADS
正常化 NI  = 8.86B CNY（TTM 干净季度）÷ 7.1 = $1.248B
正常化 EPS = 1.248B ÷ 1.63B ≈ $0.76 ~ $0.84（combine E=$0.84 用更小股数）→ 取中枢 $0.80/ADS
正常化 P/E = 现价 ÷ 0.80
g          = 第四步前瞻可持续增速 = 8%（不砍半）
合理 PE    = min(8.5 + 8, 30) = 16.5x
合理价     = 0.80 × 16.5 = $13.2
质量判定   = 好公司 × 麻烦偏一次性 → 折扣系数 ×0.67
满仓目标   = 13.2 × 0.67 = $8.8
```
