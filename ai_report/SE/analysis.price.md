**买贵了（好公司，缺好价格）——非价值席位，按成长/主题仓管理。** 与 MSFT 同档（质地好，市场没给好价格）。

**操作**
- **定性：这是成长/主题仓，不是价值安全边际仓。** $103.30 高于合理价 27%，无安全垫——回报靠"三引擎加速 + 经营杠杆继续兑现"，不靠"便宜"。仓位、止损纪律都按成长股来（波动大、估值敏感），别当价值底仓"越跌越加"
- **价值框架的"好价格"区：$51-57**（价值买点，40% 安全边际）。现价远未到，价值口径不该加
- **持有逻辑（若按成长仓持有）**：论点 = Shopee GMV +30% / SeaMoney +58% / Garena 复兴三线同时跑 + take rate 提升 + 经营杠杆。只要三引擎加速不破，成长仓可持；但清醒——为成长付了溢价，没有安全垫
- **降级/止盈**：见 base 降级信号（份额侵蚀 / 坏账 / 增速回落 + 估值不降 → 戴维斯双杀）
- **加仓纪律**：回到 $51-57（价值口径好价格）才按价值仓加；否则维持现有成长仓，不追高
