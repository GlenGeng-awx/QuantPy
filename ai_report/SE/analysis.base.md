## 二、估值分位 — P/S 近低位是"泡沫顶 de-rating 陷阱"，盈利口径仍贵

**成长股（非周期）**→ 走正常分位，但 SE 2023 才扭亏，P/E 历史短，须多口径看。

⚠️ **de-rating 陷阱（关键）**：SE 的 P/S 在 2021 曾 15-20x（纯泡沫、巨亏期）。如今 2.5x 看似"5 年低位"，但那高位本就不该存在——P/S 从 20x→2.5x 是"巨亏高估值→微利正常化"的范式切换，**不是错杀，是估值回归现实**。所以"P/S 近低"是假便宜信号；真正相关的盈利/EBITDA 口径（40x / 22x）仍贵，**不在 ≤30th 分位的合理区间**。好价格不成立。

## 五、安全边际 — 静态锚（用 min(8.5+g,30) 公式，g>21.5% 顶到 30x 封顶）

利润率改善型成长股，有正的正常化 EPS → 用公式，g 顶到 30x 封顶（价值框架不追高增长）。

```
股数      = 稀释股 ~638M（股价口径）
正常化 EPS = 正常化 NI 1.61B ÷ 638M = $2.52
g（前瞻可持续）= 顶格（近期 +47%、共识 +16-35%，均 >21.5% 阈值）
合理 PE   = min(8.5 + g, 30) = 30x（封顶）
合理价    = 2.52 × 30 = $75.6
质量判定  = 好公司（非伟大：净稀释 + FCF 含浮存）× 不算麻烦（贵）
价值买点区 = 75.6 × 0.67~0.75 ≈ $51-57
```

敏感性（前瞻利润口径交叉验）：

| 前瞻口径 | 合理价 | 说明 |
|---|---|---|
| g 顶格 → 30x 封顶（基准） | $75.6 | 高增长封顶，价值框架不追高 |
| 用前瞻 FY26 EPS ~$3.5-4（经营杠杆）× 前瞻 P/E ~31x | 仍贵 | 各口径都给不出安全边际 |
| P/S × 正常 margin 交叉验 | — | NM 6.4% 薄，P/S 2.5x 对应盈利口径仍贵 |

护栏检查：
- **高增长封顶**：g≥~22% 一律按 30x，给不出安全边际就判"买贵了"等回调 ✓
- **g 质量**：`FCF − SBC = 4.50 − 0.62 = +3.88B > 0` ✓，但 **FCF 含大量浮存，真实业主盈利更接近 NI $1.6B**，质量打折
- **回购不进 g**：SE 净稀释（股数 +14%/3yr），无缩股价值，反而是稀释拖累 ✓
- **利润率改善型**：用恢复后正常 margin 的 EPS（已用 NI），辅以前瞻 EPS / P/S 交叉验，各口径都贵
- **净现金薄托底**：~11% MCap，成长股无便宜安全垫
