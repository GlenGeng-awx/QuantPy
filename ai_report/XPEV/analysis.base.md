## 二、估值分位（盈利口径失效，用 P/S + 净现金/P/B）

盈利刚转正又季度性回亏 → P/E 失效。属**价格战 + 车型切换下行**，P/E 不可用，用 P/S 分位 + 净现金/P/B。

| 口径 | 数值 | 说明 |
|------|------|------|
| P/E TTM | 负（TTM −¥2.36） | 无意义；Q4'25 曾单季 +¥0.40 |
| P/S | 1.2x（MCap $12.51B / Rev ¥73.94B=$10.4B） | 中概车高于 LI(0.7x)/NIO(0.8x)——市场给了增长溢价 |
| EV / Sales | ~0.95x（EV ~$9.9B / Rev $10.4B） | 市场给经营业务估值（vs LI 仅 0.13x） |
| **P/B** | **~1.7x**（MCap $12.51B / 权益 ¥28.46B=$4.0B） | 高于 LI(1.2x)——账面保护较薄 |
| 净现金 / 市值 | ~18-31%（¥15-28B / MCap） | 中等厚度（远薄于 LI 88%、NIO 近 0） |

⚠️ **de-rating 判断**：XPEV 从未按盈利估过（长期亏损），P/S 1.2x 已含"增长+智驾故事"溢价。相对 LI/NIO 的 P/S 溢价合理性取决于其**营收增速最快（FY25 +87.7%）+ 盈利路径最清晰（Q4'25 已盈利）**。便宜成因是价格战 + Q1 季节性回亏引发的恐慌，非估值极端错杀——P/S 1.2x 不算深度便宜，安全边际主要来自净现金 + 增长期权。

## 五、安全边际 — 亏损股，净现金/P·S 口径（EPS 公式不适用）

min(8.5+g,30) 算不了（TTM EPS 负），按亏损/困境股护栏用净现金/P·S 口径 + 前瞻盈利：

| 锚 | 值 | 判定 |

**综合估值锚**：XPEV 无法用 EPS 公式（盈利刚转正、季度波动大）。三重口径：① **净现金 ~$2.2-3.9B（MCap 18-31%）** 提供中等下行保护；② **P/S 1.2x / EV·Sales 0.95x** 市场给了经营业务价值（区别于 LI）；③ **前瞻盈利路径清晰**（Q4'25 已盈利、FY26 共识转正）。若 FY26 稳定盈利 + 营收回增，P/S 1.5-1.7x 隐含合理价 ~$16-20。

⚠️ **净现金"地板"薄于 LI**：净现金仅 MCap 18-31%（LI 88%），下行保护主要靠"盈利路径 + 增长"而非"清算价值"。这意味着 XPEV 是"**赌盈利兑现的困境反转**"（上行更大、下行保护更薄），而 LI 是"**净现金锁底的深度困境期权**"（下行焊死、上行赌毛利修复）。XPEV 盈利路径更清晰但资产托底更薄，风险性质不同。

归类：**困境反转（止血路径清晰）— 费雪存疑，三家中最接近"好公司"边缘。** 与 LI 对比：LI 净现金厚（88%）但毛利崩塌（费雪✗更硬）；XPEV 净现金薄（18-31%）但毛利修复 + Q4'25 已盈利 + 增长最快 + 稀释最轻（费雪存疑、向好）。与 NIO 对比：XPEV 资产负债表健康得多（净现金正 vs NIO 权益近乎清零 + 慢性稀释）。**不属"好公司遇一次性麻烦"的框架猎物（那是 ADBE），是"盈利路径清晰的困境反转"**——适合承担不对称风险者：**≤小仓博盈利兑现 + 增长**，非价值席位。质量与止血确定性均优于 LI/NIO，但净现金托底最薄，需盈利兑现来验证。

**升级信号（转"好公司"重看）**：连续 2 季恢复单季盈利（回 Q4'25 轨道）+ 车辆毛利回 ≥15% + FCF 剔浮存后仍正 → 脱离"期权"按成长反转标的重估（可给 P/E 公式）。
**降级信号（避）**：Q2 反弹不及指引（交付 <10 万）+ 毛利回落 <18% + 净现金持续消耗 + 恢复股权融资稀释 → 盈利路径证伪，回避。
