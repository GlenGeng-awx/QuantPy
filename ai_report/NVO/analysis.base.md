## 二、估值分位（区间结构 + 同业 + de-rating）

**非周期股**（专利药，盈利不随商品价摆动）→ 走正常分位。GLP-1 有产品周期但非商品周期。

⚠️ **de-rating 陷阱（NVO 特别严重）**：历史 36-41x 是 2021-23 GLP-1 狂热期的**成长泡沫溢价**（当时增速 20-30%+、市场按 "无限 TAM" 定价）。如今增速切到 2026 **负增长**、2027 才恢复双位数 → 从 40x 掉到 12x **大部分是合理范式切换**（"高增长小竞争 -> 双寡头分食 + 降价" 的 de-rating），**不全是错杀**。这与 ADBE（89% GM 垄断、增速仍 12%）性质不同——NVO 的便宜里含"真实竞争恶化"成分，须打折看。安全边际不能只看"vs 历史 40x 便宜了 70%"。

### SCORECARD 背离检验（机械信号 = 假设 → 读原始财报证实/推翻 → 方向）

SCORECARD 是报案人非评委。逐维度把机械信号与原始财报现实对照，gap 方向即错杀/陷阱藏身处：

| 维度 | SCORECARD 说（机械信号） | 读原始财报证实/推翻 | 背离方向 → 机会/陷阱 |
|------|----------------|----------------|----------------|
| **Income TTM/5Q 满分（100/100）** | 机械看**漂亮**：TTM EPS +19%、Q1'26 EPS +67% YoY、OpInc +54% | **部分推翻（垫高假象）**：满分几乎全靠 Q1'26 单季暴涨，而 Q1'26 是 **vs 极弱基数 Q1'25**（去年降价/去库存低点），且 2026 全年公司指引仍是**负增长 -4~-12%**（前高后低）。TTM 还含 DKK 6.08B 证券出售一次性收益（Gain On Sale）垫高 NI | **⚠ 陷阱方向（机械漂亮、实则垫高）**：满分掩盖"全年下滑年"现实，别把 Q1 反弹当持续斜率。正常化 NI 117.2B vs 报告 122.0B（-4%） |
| **Income 3yr=80（OpInc -0.5%、GM 84.6%→81.0%）** | 机械看**丑**：OpInc 3 年近乎持平、GM 下滑扣分 | **证实（真实侵蚀）**：GM 从 84.6%→81.0% 是**真的**——美国 MFN 降价 + 中/巴/加专利到期 + LLY 竞争定价压力，不是一次性项误伤。OpInc 停滞也真实（增长引擎从 20%+ 掉到 0） | **✓ 陷阱/合理 de-rating 方向（机械丑、财报证实也在恶化）**：这一格是**结构性成分的证据**，市场大概率已正确定价（40x→12x 的 de-rating 有一部分合理），alpha 少 |
| **BS 3yr=32（D/E 0.20→0.63）** | 机械看**丑**：D/E 飙升 3 倍、Cash/Debt 仅 0.15x 严重扣分 | **半推翻（并购+扩张举债，非经营失血）**：D/E 飙升主因 = Catalent 收购（2024 净收购 DKK -82B）+ 产能扩张举债，是**主动加杠杆买产能/买 CDMO**，非亏损被动举债。利息覆盖 58.9x 极健康。**但**净负债 125B DKK 是真的、无净现金托底也是真的 | **中性偏陷阱：机械丑有真因（无资产托底），但不是"经营崩塌式"举债**。区别于价值陷阱（那是借钱续命）；这里是借钱扩产的赌注——赌产能能变现 |
| **CF FCF 3yr -58%（70B→31B）** | 机械看**丑**：FCF 腰斩 | **半推翻（CapEx 驱动，非 OCF 恶化）**：OCF 3yr 仅 -1.5%（118B 稳），FCF 崩是 CapEx 从 39B→90B（Kalundborg+Clayton+Chartres 产能）吞噬。**是"用现金换未来供给"的资本开支台阶，非经营失血** | **✓ 部分错杀方向（机械丑、OCF 证实经营未恶化）**：FCF -58% 吓人但 OCF 稳——**若产能如期变现，2026 CapEx 见顶回落 DKK 55B → FCF 恢复**。这是暂时性一次性台阶。但前提是 GLP-1 需求兑现（LLY 若赢家通吃则变烧钱） |

**综合背离读数**：NVO 的 SCORECARD 背离**不是单向的**——Income 3yr（GM 侵蚀）是**机械丑+财报证实=真恶化**（陷阱/合理 de-rating 方向），而 CF FCF -58% 是**机械丑+财报推翻=错杀**（CapEx 台阶非失血，一次性方向）。**这正是 NVO"混合伤口"的量化证据**：一半结构（竞争/降价压 GM，市场对了）、一半一次性（CapEx 周期 + Q1 弱基数，市场过度反应）。与第二步屏幕 P/E 背离（12x 便宜里含真实竞争恶化、非纯错杀）同向印证——**机会打折，非干净买点**。

### 伤口模式

3yr OpInc 持平 + GM 下滑 + FCF -58% + 2026 指引下滑 = **混合伤口**：一部分是产能投资周期（一次性台阶，2026 CapEx 见顶）+ 谷底利润年（暂时），一部分是 **LLY 竞争 + 降价的结构性压力**（GM 从 84.6%->81% 有真实侵蚀迹象）。非纯情绪伤口（区别于 ADBE），也非纯结构崩塌——**居中，属"一般麻烦、偏一次性但存疑"**。上表 SCORECARD 背离检验量化了这个"混合"：GM 侵蚀那格机械丑+财报证实（真结构），FCF 那格机械丑+财报推翻（一次性 CapEx 台阶）。

### 护城河 — 宽但正在被侵蚀（关键区别于 ADBE 的"纹丝不动"）

- 壁垒类型：semaglutide 品牌（Ozempic/Wegovy 全球认知）、肽类生物制药**生产工艺 + 产能**壁垒（GLP-1 产能是全球瓶颈，NVO+LLY 双寡头）、专利、监管审批
- 份额趋势：⚠️ **正在流失**——LLY 占美国 GLP-1 组合 **60.1%** vs NVO **39.4%**，LLY 在美国外也已反超；tirzepatide（Zepbound）减重效力 25.5% > CagriSema 23%
- 威胁：**LLY orforglipron 口服 GLP-1**（Phase 3 达标、全球提交中，直接威胁 NVO 口服 Wegovy）；专利悬崖（semaglutide 化合物专利 ~2031-2032 起）；美国 MFN 降价（低-中双位数%）；compounding（复方药）灰色竞争

护城河仍**宽**（双寡头之一、产能壁垒、TAM 巨大），但不是 ADBE 式"格式锁定不可撼动"——是"两强分食且对手在领先"的动态战场。

## 五、安全边际 — 静态锚（合理价 / 满仓目标）

一般/成长股，有正常化 EPS。**全程 ADR/USD 股价口径**（1 ADR = 1 B 股）。g 不砍半（保守性已在第四步做）。

```

g（前瞻,不砍半）= 8%          # 第四步产出:2026 负增长 + 2027 恢复双位数,穿越中枢;LLY 抢份额封上限,口服+Medicare 撑下限
合理 PE   = min(8.5 + 8, 30)  = 16.5x
合理价    = 4.17 x 16.5        = $69
质量判定  = 伟大 x 一般麻烦(偏一次性但存疑) -> 折扣系数 x0.67

```
