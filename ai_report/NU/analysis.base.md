### 伤口模式

## 五、安全边际 — 静态锚（合理价 / 满仓目标）

高增长盈利银行，g 不砍半（保守性已在第四步取放缓值）。正常化 EPS = TTM $0.65（Unusual 近零；Q1'26 低税率是唯一扰动，用 TTM 已平滑）。

```
正常化 EPS = $0.65
g（前瞻，不砍半）= 25%             # 5yr 共识 ~25.7%,取放缓值;g≥22% 触封顶
合理 PE   = min(8.5 + 25, 30) = 30x（封顶）   # 价值框架不追高增长
合理价    = 0.65 × 30            = $19.50
质量判定  = 好公司(高增长银行) × 一般麻烦(存疑) → 折扣系数 ×0.67

```
