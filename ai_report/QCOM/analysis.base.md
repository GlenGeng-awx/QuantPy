## 二、估值分位（区间结构 + 同业 + de-rating）

**半导体 = 周期股**——但 QCOM 特殊：约 1/3 利润来自 QTL 授权（近似年金、跨周期稳定），2/3 来自 QCT 芯片（随手机/物联网周期摆动）。故 P/E 分位不像纯周期矿业那样极端误导，但仍需谨慎：手机出货低谷时 QCT 利润被压，P/E 偏高≠贵。当前处于**苹果 modem 流失的下行叙事** + 手机温和周期，宜用中周期正常化盈利看。

### 利润表 — 逐季拆解（关键：一次性税项污染 EPS）

```
        Rev      GM%    OpInc   OpM%    税项      NI       dilEPS   dilShares
Q1'25  10.98B  55.0%  3.12B   28.4%   293M    2.81B    2.52     1.11B
Q2'25  10.37B  55.5%  2.76B   26.6%   286M    2.67B    2.43     1.10B
Q3'25  11.27B  55.4%  2.96B   26.3%   6.09B   -3.12B  -2.89     1.08B   <- 巨额税费
```

- **GM 稳定 53.8-55.5%**——芯片级毛利，远低于软件；QCT 端有周期性小幅波动，QTL 授权 GM 极高但被 QCT 稀释
- **⚠️ EPS 被税项严重扭曲**：Q3'25 单季 −$2.89（$6.09B 税费）、Q1'26 单季 +$6.88（$5.14B 税收利得）。这两笔互相部分抵消但**使 TTM GAAP EPS $9.30 完全失真**。TTM NI $9.92B 含 Q1'26 的 $5.14B 一次性税收利得 → 正常化必须剔除
- **营收放缓已现**：Q1'26 营收 $10.60B、−3.5% YoY；OpInc −26% YoY、EBITDA −23% YoY = 手机周期回落 + 苹果流失前兆
- **缩股扎实**：dil shares 1.11B→1.07B（真回购缩股）

**正常化 EPS 推导**（剔一次性税项，用经营盈利能力）：
```
TTM Operating Income          = $11.39B
− 利息                        = $0.68B
= 经营税前（剔波动 other/税）  = $10.71B
× (1 − 13% 正常化税率)         = $9.32B 正常化 NI
÷ 1.08B 稀释股                = 正常化 EPS ≈ $8.60
```
→ **正常化 GAAP P/E（@$177）≈ 20.6x**，比屏幕 19x 更贵，非更便宜。

### 护城河 — 分裂（授权宽、芯片窄）

- **QTL 授权（宽）**：CDMA/OFDMA/5G 标准必要专利池，每台安卓设备抽成，跨厂商、跨周期年金式高毛利。真护城河。
- **QCT 芯片（中→窄）**：Snapdragon 安卓高端 SoC 领先，但**苹果自研 modem 已证明大客户可垂直整合出走**（切换成本被大客户攻破）；安卓端联发科竞争、三星自研 Exynos。
- 份额趋势：安卓高端稳固；苹果份额归零；汽车份额上升（Snapdragon Digital Chassis）。
- 威胁：客户垂直整合（苹果模式或蔓延）、手机成熟、无 AI 数据中心敞口（AVGO/NVDA 吃走 AI 红利）。

## 五、安全边际 — 静态锚（合理价 / 满仓目标）

半导体周期股但含稳定授权年金，用正常化盈利 + g 不砍半。正常化 EPS 剔一次性税项 = **$8.60**（见第三步推导）。

```
正常化 EPS = $8.60                       # 剔 Q3'25/Q1'26 一次性税项，用经营盈利能力
g（前瞻，不砍半）= 5%                     # 汽车/IoT 高增长被苹果流失 + 手机成熟对冲后的保守净增速
合理 PE   = min(8.5 + 5, 30) = 13.5x
合理价    = 8.60 × 13.5              = $116
质量判定  = 好公司 × 麻烦一般偏结构      → 折扣系数 ×0.67
满仓目标  = 116 × 0.67              = $78
```

敏感性（g 口径 + EPS 口径）：

| 前瞻增速 g | 合理 PE | 合理价（EPS $8.60） | 满仓目标（×0.67） |
|---|---|---|---|
| 3%（若苹果流失快于汽车补） | 11.5x | $99 | $66 |
| 5%（基准，保守净增速） | 13.5x | $116 | $78 |
| 6%（汽车/IoT 顺利吸收） | 14.5x | $125 | $84 |

护栏检查：
- **回购不进 g**：g=5% 用业务增长，缩股价值体现在 EPS 基数 + 安全边际，不抬合理 PE ✓
- **g 质量**：`FCF − SBC = +$9.43B > 0` 且 3 年增长 ✓
- **一次性 vs 正常化 EPS**：TTM GAAP EPS $9.30 被 Q1'26 $5.14B 一次性税收利得垫高 → 必须剔除，用 $8.60。**这是屏幕 P/E 低估贵的典型**（一次性收益垫高利润，正常化后更贵）✓
- **周期护栏**：QCOM 授权年金削弱周期误导，但 QCT 仍周期；正常化用中周期口径，未在周期底部 ✓
- **数据中心作上行期权**：FY2028 潜在贡献不计入 g 基线 ✓
