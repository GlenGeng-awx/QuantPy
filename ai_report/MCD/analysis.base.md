**质量判定**：**伟大（瑕疵在杠杆）**（GM 57.4% 稳、NM 31.6%、FCF yield ~3.7%、`FCF−SBC = +$6.86B`、全球 QSR 最宽品牌 + 95% 加盟轻资产版税护城河、48 年股息贵族、真缩股 −3.5%/3yr；**瑕疵：负账面权益 + 净债 $53.6B，无净现金托底**）× 麻烦 **存疑（GLP-1 半结构 + 消费周期性）**　→　折扣系数 **×0.67**（伟大遇存疑麻烦降至好公司+一般档）

**核心锚（静态）**：正常化 EPS **$12.35** → g **7%**（剔回购）→ 合理 PE **min(8.5+7, 30)=15.5x** → **合理价 $189** → 满仓目标 **$127**（×0.67）

> **框架张力（先声明）**：MCD 是"低增长 + 超高质量 + 防御溢价"典型。价值框架把倍数封在 min(8.5+g,30)=15.5x，质量只进折扣系数、不进倍数，因此系统性判 MCD 这类防御复利股"买贵了"——除非它真崩到 15.5x。这是格雷厄姆不为 7% 增长付 22x 的设计使然，非 bug。想要 MCD 级 durability 的债券替代持有者，本框架不覆盖。

## 二、估值分位（区间结构 + 同业 + de-rating）

**非周期股**（QSR 版税模型），走正常分位。当前 P/E 与所处分位见 price 文件。

### MCD 自身 5 年区间

| 口径 | 数值 | 分位 | 来源 |
|------|------|------|------|
| 屏幕 P/E TTM | ~23x（E=12.13） | 自身低端 ~25-35% | combine |
| 前瞻 P/E | ~20x | 低端 | 分析师共识 |
| 5 年 P/E 区间 | ~20-27x，中位 ~24-25x | 当前落低端 | [MarketBeat](https://www.marketbeat.com/stocks/NYSE/MCD/forecast/) |

当前 ~23x 落自身 5 年低端（~25-35 分位），未硬达 ≤30% 但接近。**但——低端 ≠ 绝对便宜。**

⚠️ **de-rating 陷阱**：MCD 历史 24-27x 高分位部分来自**低利率时代"债券替代品"溢价**（防御 + 稳定股息 + 低 beta）。利率正常化 + 增速放缓 + GLP-1 结构 overhang 后，回落到 ~23x 有一部分是**合理的范式切换，不全是错杀**。不能拿 25x 当"正常锚"论证"回到 23x 就便宜"。真正的绝对锚是"为 6-8% 增长该付多少倍"——见第五步（15.5x）。

### 利润表 — 逐季拆解

```
        Rev     GM%    OpInc   OpM%    NI      NM%
Q1'25  5.96B  56.0%   2.65B  44.5%   1.87B  31.4%
Q2'25  6.84B  57.9%   3.26B  47.7%   2.25B  32.9%
Q3'25  7.08B  57.9%   3.32B  46.9%   2.28B  32.2%
Q4'25  7.01B  57.5%   3.16B  45.1%   2.16B  30.8%
Q1'26  6.52B  55.9%   2.88B  44.2%   1.98B  30.4%
```

- **GM 55.9-57.9% 极稳**——加盟版税模型的美妙：收入大头是特许权使用费 + 租金，GM 天然高且稳。年度 57.1→56.8→57.4%
- **OpM 44-48%**（管理层 2026 指引 45-47%），QSR 顶级
- **Q1'26 环比走软**：营收 6.52B（季节性淡季）、GM 55.9%、NM 30.4% 双降——对应美国客流 −2.3%（两年最大降幅）、消费者缩减开支。**是当前回撤的直接诱因**
- 利润增长质量：OpInc +4-5% 由系统销售 + 单位扩张驱动（接新水），非砍费用；但增速温和（非高成长）
- 正常化：Total Unusual 仅 −$193M（TTM），Normalized NI $8.83B vs GAAP NI $8.68B，**盈利基本干净**

### 现金流 — 优质现金牛

| 项目 | TTM | 说明 |
|------|-----|------|
| OCF | 10.54B（OCF/NI 1.21） | 含金量高（D&A + 递延税） |
| FCF | 7.04B（yield ~3.7%@MCap ~199B） | 稳定；CapEx/OCF 33%（含开店） |
| SBC/Rev | 0.7%（极低） | 蓝筹典范 |
| 股东回报 | 分红 5.17B + 回购 1.98B，Net Return **+6.97B**（yield ~3.5%） | 分红为主（股息贵族 48 年连增），SBC 仅 180M → 真缩股 |
| 缩股 | 稀释股 741→715M（−3.5%/3yr） | 真缩股 |

`FCF − SBC = 7.04 − 0.18 = 6.86B > 0`（成长估值合格，现金创造质量极高）。**但 FCF yield 仅 ~3.7% = 优质但不便宜的量化体现。**

### 资产负债表 — 激进资本结构（关键风险维度）

| 项目 | 值 | 解读 |
|------|-----|------|
| 现金 | 1.17B | 极少（现金牛无需囤现金） |
| 总债 | 54.88B（+ 租赁 14.78B） | 高杠杆 |
| **净债** | **~53.6B** | **无净现金托底**——下行无资产缓冲（价值口径零安全垫） |
| **股东权益** | **−$1.79B（负）** | 数十年回购（Treasury $79.32B）+ 分红超留存所致，**P/B 无意义** |
| Net PPE | 42.76B（大量优质地产） | **真实资产底**：MCD 半个地产公司，负账面权益被低估的地产 mask |
| 利息覆盖 | 7.9x | 可服务，非偿债风险 |

**负权益辨析**：不是资不抵债，是"现金牛主动把 >100% 利润通过回购 + 杠杆返还股东"的财务工程（同 SBUX/HD/AZO）。稳定版税现金流 + 优质地产支撑得起。**但对价值框架的含义**：① P/B 失效；② **无净现金/清算价值托底**——不像 TCOM/PDD 有净现金锁下行，MCD 下行只能靠盈利与地产估值支撑，买贵了就得硬扛回撤。EV/EBITDA ~17x 是比 P/E 更诚实的估值镜。

### 伤口模式

Income 88 / CF 优质，非经营伤口。回撤源于**消费周期性走软（客流 −2.3%）+ GLP-1 结构 overhang 情绪 + 估值 de-rating**三者叠加。基本面完好（SSS 仍正 +3.9%），属"温和情绪/周期伤口 + 合理估值回归"，非结构性恶化。

## 四、增长前瞻 / 护城河 / 管理层 / 消息面

### 增长前瞻（产出第五步 g）

| 口径 | 数值 | 说明 |

| 近期 SSS | Q4'25 US +9%（$5 套餐拉动）→ Q1'26 US +3.9%、国际发展许可 +3.4% | 正但放缓 |
| 单位扩张 | 2026 计划新开 ~2,600 店（约 43,000 基数 → +6% 单位增长） | 结构性增长引擎 |

| 前瞻共识 | 营收 CAGR ~5-6%、EPS ~7-9%（含回购） | 温和成长 |

**产出前瞻可持续增速（剔回购）≈ 7%**：单位增长 +6% + SSS +3-4% 驱动系统销售 → 加盟版税/收入端 ~5-7%（近期客流走软下调）。EPS 更高的部分靠回购，**护栏：回购不进 g**，故 g 取业务口径 7%。

### 护城河 — 全球 QSR 最宽，但 GLP-1 是长期新变量

- 壁垒类型：**全球最强 QSR 品牌** + 规模采购成本优势 + 黄金地段地产 + 加盟版税模型（轻资产、高 ROIC）+ Drive-thru/App/配送网络。43,000+ 门店，护城河极宽
- 份额趋势：全球门店持续扩张，价值菜单（$5 套餐）稳住低端客流
- **威胁**：① **GLP-1 减肥药**——CEO 称"暂无实质影响"但承认"随渗透率上升消费者行为会变"，长期结构性削减快餐热量摄入（MCD 以高蛋白菜单 + GLP-1 友好选项应对）；② 低收入消费者被通胀挤压（周期性）；③ 快餐竞争 + 健康化趋势。**GLP-1 是护城河没被攻破、但需求端天花板被质疑的新型威胁**——半结构、存疑

### 熊 / 牛逻辑

**熊市逻辑**：增速温和（营收 +3.7%、EPS ~7% 含回购）却要 ~23x P/E——为防御性溢价买单，价值框架不认；客流 −2.3% + 低收入消费者疲软；GLP-1 长期削减快餐需求天花板（结构存疑）；**负权益 + 净债 \$53.6B 无托底**，下行硬扛；de-rating 未必结束（利率 + 增速 + GLP-1）；EV/EBITDA ~17x 对 7% 增长偏贵。

**牛市逻辑**：全球 QSR 最宽护城河 + 版税轻资产模型（OpM 45-47%、ROIC 极高）；48 年股息贵族 + 真缩股，股东回报机器（Net Return yield ~3.5%）；单位扩张 +6% + 价值菜单稳客流 = 温和但可靠的复利；\$43B 优质地产底被负权益 mask；防御属性经济下行更抗跌；GLP-1 影响"暂未显现"且主动适配。**但——牛论都是"质量与 durability"，不解决"当前价对 7% 增长无安全边际"。**

## 五、安全边际 — 静态锚（合理价 / 满仓目标）

新口径链条（g 不砍半、g 剔回购）：

```
正常化 EPS = 正常化 NI ÷ 稀释股数 = 8.83B ÷ 715M = $12.35   （GAAP 稀释 $12.13，几乎一致，盈利干净）
g          = 第四步业务增速（剔回购）               = 7%
合理 PE    = min(8.5 + 7, 30)                     = 15.5x
合理价     = 12.35 × 15.5                          = $191 → 取 $189
质量判定   = 伟大（瑕疵在杠杆）× 麻烦存疑            → 折扣系数 ×0.67
满仓目标   = 189 × 0.67                            = $127
```

敏感性（对 g 与是否给防御溢价）：

| 情景 | g | 合理 PE | 合理价 | 满仓目标（×0.67） |
|---|---|---|---|---|
| 保守（近期客流走软） | 6% | 14.5x | $179 | $120 |
| 中枢（基准） | 7% | 15.5x | $189 | $127 |
| 乐观（单位扩张加速） | 8% | 16.5x | $204 | $137 |
| 【框架外参照】市场防御溢价 | — | ~22x（历史锚） | ~$272 | — |

护栏检查：
- **回购不进 g**：g=7% 用系统销售 + 单位增长，EPS 里的回购贡献不抬合理 PE ✓。MCD 是"净利温和 + 缩股撑 EPS"的典型
- **g 质量**：`FCF − SBC = +6.86B`、3 年 FCF 稳增 ✓
- **高增长封顶**：g=7% << 22%，未触 30x 封顶 ✓（MCD 反而低增长端，合理 PE 仅 15.5x）
- **无净现金锚**：净债 \$53.6B、负权益 → 无下行资产托底（区别于 TCOM/PDD 净现金），买贵的回撤只能硬扛；唯一软垫是 \$43B 优质地产（非流动、非清算口径）
- **框架张力**：见头部声明——质量只进折扣、不进倍数，系统性判 MCD "买贵了"，除非崩到 15.5x
