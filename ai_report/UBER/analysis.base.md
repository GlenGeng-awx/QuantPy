## 五、安全边际 — 静态锚（合理价 / 满仓目标）

**利润被一次性项扭曲型**——GAAP NI 被股权 MTM + 一次性递延税双料扰动，须用**经营口径正常化 EPS**。

```
正常化 EPS 推导（剔 MTM + 一次性税，用经营口径）：
  Op Income TTM        = $6.26B
  - 利息               = $0.44B
  = 核心税前           ~ $5.82B
  x (1 - 24% 正常税率)  = $4.42B  正常化 NI
  / 稀释股数 2.11B      = $2.10   正常化 EPS

g（前瞻，不砍半）= 14%（出行放缓+外卖驱动+长期递减，取保守中值）
合理 PE   = min(8.5 + 14, 30) = 22.5x
合理价    = 2.10 x 22.5        = $47.25
质量判定  = 好公司 x 麻烦存疑/一般 -> 折扣系数 x0.67
满仓目标  = 47.25 x 0.67       = $32
```

敏感性（g 口径）：

| 前瞻增速 g | 合理 PE | 合理价 | 满仓目标（×0.67） |
|---|---|---|---|
| 10%（长期共识，保守） | 18.5x | $38.9 | $26 |
| 14%（近期，基准） | 22.5x | $47.25 | $32 |
| 18%（历史 CAGR，乐观） | 26.5x | $55.7 | $37 |

交叉验（FCF 口径，避免正常化 EPS 争议）：FCF $9.8B，@ 合理 25x FCF = $245B 合理市值 ÷ 2.11B 股 ≈ $116/股。**FCF 口径远高于 EPS 口径**——因 FCF 含营运资本浮存（应付膨胀 +$2.96B）与 D&A/税盾，用它作锚偏乐观。取两者保守偏 EPS 口径 $47.25 为基准，但 FCF 口径提示下行有限、现价并非荒谬（软估算详见综合）。

护栏检查：
- **回购不进 g**：g=14% 用业务增长，缩股价值在安全边际+EPS 基数 ✓
- **g 质量**：`FCF − SBC = +$7.94B > 0` 且 3 年增长 ✓
- **利润被麻烦压低 → 用正常化经营 EPS**：已剔 MTM+一次性税 ✓
- **AV 不确定作下行风险（非上行期权）**：结构性威胁，不给护城河溢价 ✓
- **FCF 口径含浮存打折**：应付膨胀撑起的现金随增速放缓会回流，作锚打折 ✓
