## 五、安全边际（亏损股，现价代入 P/S / 流动性锚）

EPS 公式不适用（深负）。按 base 的 P/S / 流动性 / 无净现金地板口径定位：

```
净现金        = −1.75B（净债）        → 无下行硬地板
P/B           = 25.02 / 4.40 = 5.7x   → 远离账面，无清算托底
P/S TTM       = 4.5x                   → 相对已盈利车企数倍溢价

有效流动性    = 4.83B + VW ~2.0B + DOE 4.5B ≈ ~8B vs 年烧 ~3B → ~2.5 年（含条件性外部钱）
现价 $18.58   → 无安全边际，买贵了
```

**三要素全 ✗ → 困境反转期权，非最佳买点、非价值买入。现价 $18.58 买贵了。** 与 LI（净现金 88% 硬地板的期权）不同，RIVN 地板软（净债 + 外部条件性输血），风险更高。

**操作**
- **不入价值池。** 现价无安全边际（P/S 4.5x、已反弹 +61%、无净现金地板）
- 若愿承担不对称投机：仅在 **现价回落至 ~$11（前瞻 P/S <2x，近 52 周低点）** 且 GM/R2 反转持续，才 ≤2-3% 极小仓，非价值席位
- 加仓：不适用（非建仓标的）
- 止损/回避触发：GM 再度转负 + 现金 <$3B 被迫大额增发（>10% 稀释）+ R2 爬坡显著不及 → 滑向价值陷阱，回避
- 升级触发（价+基本面）：连续 2 季汽车 GM ≥15% + FCF 接近转正 + R2 季交付破 15K + 无需再增发 → 按反转标的重估
