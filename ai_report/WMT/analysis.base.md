| P/E TTM | 39.4x（E=2.84） | 对 4-5% 增长的零售商极高 |

⚠️ **de-rating 陷阱反向**：WMT 不是被 de-rate，是被 **re-rate 上修**——市场把 Walmart Connect 广告（高margin）+ 会员（Walmart+/Sam's）+ 电商盈利化当"科技/广告"故事给溢价，从历史 ~22x 推到 39x。这是**为"防御 + 新margin引擎"付满价**，非错杀。同业：COST ~50x（更贵）、TGT ~13x、KR ~12x——WMT 在传统零售里最贵，仅低于 COST。

## 四、增长前瞻 / 护城河 / 管理层

- **g**：营收 +4-5%、EPS +7-13%（含回购）。业务 g（剔回购）≈ **6%**：电商 + Walmart Connect 广告（高margin，高增长）+ 会员 + 自动化提效。近期客流稳、消费降级利好（低价定位受益通胀）
- **护城河 — 宽**：全球最大零售商、规模采购成本优势、密集门店 + 配送网（当日达）、EDLP 心智、Sam's 会员。广告/媒体网络是新高margin护城河
- **管理层**：Doug McMillon（长期，2026 有交接讨论）；资本配置稳健，48 年股息贵族；近年重投自动化/电商
- **消息面**：广告业务（Walmart Connect）+ 会员双位数增长，市场按"零售 + 广告 + AI"给溢价；关税/供应链是宏观变量

**熊**：39x P/E 买 3.1% NM、4-5% 增长的零售商 = 为防御 + 广告故事付满价；净债无托底；CapEx 重吞 FCF；一旦广告/会员增速不及预期，估值下修空间大。
**牛**：全球最强零售护城河 + 广告/会员高margin再加速（margin mix 上行）；防御属性经济下行抗跌；48 年增息 + 真缩股；低价定位受益消费降级。

## 五、安全边际 — 静态锚

低增长防御股，g 不砍半。正常化 EPS = TTM 稀释 $2.84（干净）。

```
正常化 EPS = $2.84
g（业务，剔回购）= 6%
合理 PE   = min(8.5 + 6, 30) = 14.5x
合理价    = 2.84 × 14.5       = $41
质量判定  = 好公司 × 无麻烦    → 折扣系数 ×0.67
满仓目标  = 41 × 0.67          = $27.6
```

护栏：
- **回购不进 g**：g=6% 用业务增长（广告/电商/会员），回购另计不抬 PE ✓
- **框架张力（同 MCD）**：低增长防御复利股，框架封 PE 14.5x、质量只进折扣不进倍数 → 系统性判"买贵了"，除非崩到 14.5x。市场的 39x 是防御 + 广告溢价，本价值框架不认
- **无净现金托底**：净债 $41B，买贵无资产缓冲
