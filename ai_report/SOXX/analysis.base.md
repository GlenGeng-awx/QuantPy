⚠️ **周期股 P/E 反读陷阱**：半导体盈利随存储/GPU 周期剧烈摆动。**顶部 EPS 高 → 屏幕 P/E 看似"仅 30x 不算贵"，实为周期顶部**（低 P/E ≠ 便宜）。正确读法用中周期正常化：把峰值 EPS 打回跨周期中枢后，真实 P/E 远高于 30x。屏幕上 30x + 近 ATH 双证据 → 高分位。

⚠️ **de-rating 反向**：2026 AI 景气把半导体从"周期股"重估为"AI 长期成长股"，多头认为该给成长溢价。但价值框架下这正是**范式切换的风险点**——一旦 AI 资本开支节奏放缓，峰值 EPS + 高倍数双杀。

## 三、结构（周期性 / 集中度 / 驱动）

- **周期性**：极强。半导体是典型周期股（存储 MU、代工 TSM、设备 AMAT/LRCX 皆随资本开支周期摆动）。CLAUDE.md 周期股规则直接适用：不能拿屏幕 P/E 当贵贱信号。
- **集中度**：前十大 60%，前五 ~36% 集中在 AI GPU/存储龙头。**非分散篮子**——买 SOXX ≈ 加杠杆押注 AI 芯片资本开支周期不破。
- **驱动**：单一驱动 = **AI 数据中心资本开支**（NVDA/AVGO/AMD GPU + MU HBM + ASML/AMAT 设备）。驱动高度同向 → 篮子内相关性极高，回撤时齐跌，分散效果弱。

## 四、宏观前瞻（板块周期位置、熊/牛）

**板块周期位置**：**AI 上行周期顶部区域**。SOXX YTD +89%、过去 1 年总回报 +152%、距 52 周高点 $655.95 仅 ~13%。RSI 在价格创新高时反而走弱（背离，历史上是警示信号）。估值+价格双高。

**熊市逻辑**：
- 周期顶部特征齐全——峰值 EPS + 高倍数 + 近 ATH + 动量背离
- 单一驱动（AI capex）若见顶/放缓 → 峰值 EPS 与倍数双杀（半导体历史回撤常 -40~60%）
- 集中度 60% 无分散护身；前十大多只（MU/AVGO/AMD/ASML）我们已独立判"贵"
- 盈利收益率 ~3.3% < 无风险利率，风险溢价为负

**牛市逻辑**：
- AI 资本开支若为多年结构性超级周期（非传统 2-3 年周期），则"峰值 EPS"其实是新常态起点，30x 不贵
- 半导体自主可控/AI value chain 全覆盖，长期需求上台阶
- 若周期不破，动量+盈利上调可续（2026 YTD +89% 即此逻辑演绎）
