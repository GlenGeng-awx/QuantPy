## 五、安全边际 — 静态锚（合理价 / 满仓目标）

正常化：**GAAP NI $247M 被 NOL 免税（实际税率 6.8%）虚高，还原到常态 21% 税率**：正常化 NI = 税前 $265M ×（1−21%）= **$209M** → 正常化 EPS = 209M/178M = **$1.18**。

```
正常化 EPS = $1.18（21% 常态税率）
g（前瞻，不砍半）= 9.5%（指引 9-10% + 近期 YoY 11%，取保守）
合理 PE   = min(8.5 + 9.5, 30) = 18.0x
合理价    = 1.18 × 18.0 = $21
质量判定  = 平庸/好公司边缘 × 无麻烦（本就不便宜）→ 折扣系数 ×0.60
满仓目标  = 21 × 0.60 = $13
```

敏感性（g 口径）：

| 前瞻增速 g | 合理 PE | 合理价 | 满仓目标（×0.60） |
|---|---|---|---|
| 9.5%（指引，基准） | 18.0x | $21 | $13 |
| 16%（历史 CAGR，乐观） | 24.5x | $29 | $17 |
| FCF 口径交叉验：FCF−SBC $363M/178M=$2.04/股 × 18x + 净现金 $12/股 = ~$49 | — | ~$49 | — |

**即便最乐观的 FCF−SBC + 净现金口径 ~$49，也远低于现价 $141。** 主锚合理价 $21。

护栏检查：
- **回购不进 g**：g=9.5% 用业务增长 ✓；且回购 < SBC，无缩股价值可加 ⚠️
- **g 质量**：`FCF − SBC = +$363M > 0` 勉强过线，但 **Net Return −$212M（SBC 未被回购抵消）** = 成长质量差，SNOW 类边缘 ⚠️
- **NOL 免税必还原**：GAAP 税率 6.8% 不可持续，正常化 21% 后 P/E 从 101x → 120x（屏幕已贵，正常化更贵）✓
- **净现金托底薄**：仅 $2.18B（9% MCap），Goodwill 57% 有减值风险，托底质量中等 ✓
