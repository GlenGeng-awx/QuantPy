## 二、估值分位（区间结构 + 同业 + de-rating）

**先判周期股**：Ericsson 是**强周期股**——盈利随运营商 5G/RAN capex 周期剧烈摆动（OpInc 2022 26.8B → 2024 谷底 6.2B → TTM 26.2B SEK，三年 4x 摆动）。→ P/E 分位会误导（谷底利润低、P/E 高=买点；顶部 P/E 低=卖点）。改用中周期正常化 P/E + P/B + 净现金判贵贱。

| 屏幕 P/E TTM | 12.3-14.0x | [macrotrends](https://www.macrotrends.net/stocks/charts/ERIC/ericsson/pe-ratio)、[stockanalysis](https://stockanalysis.com/quote/sto/ERIC.B/statistics/) 2026-06 |

**周期定位判断**：TTM adj EBITA margin ~14%（Q1'26 11.3%），管理层目标 **15-18%**——说明当前**利润率尚未到中周期中枢**，仍在恢复途中（非顶部）。但屏幕 P/E 低是**一次性处置收益**造的假象，非周期顶部低 P/E。P/B ~1.1x 非极端底部（周期底 P/B <1.0）。**当前不在明确周期底部，也无处置收益剔除后的安全边际**。

### de-rating 判断

Ericsson 长期就是低倍数周期股（历史中位 ~13-15x 正常化），**当前无 de-rating 错杀**——它一直不贵也不便宜。屏幕 12-14x 的"便宜"是处置收益制造的假象，正常化 17x 反而略高于历史中枢。

## 四、增长前瞻 / 护城河 / 管理层 / 消息面

### 增长前瞻（产出第五步 g）

| 口径 | 数值 | 说明 |
|------|------|------|
| 历史 3 年营收 CAGR | **−4.5%/年**（271.5B 2022 → 236.7B 2025 SEK） | 下滑（RAN capex 见顶回落 + 汇率） |
| 近期季度 YoY | Q1'26 reported −10.3%，organic **+6%** | 汇率拖累大 |
| RAN 市场 | 管理层：**"flattish"**，北美 2026 偏软 | 成熟市场，5G 建设高峰已过 |
| 增长引擎 | 企业 5G / Network APIs（三收入流周期论） | 尚小、期权性 |
| 前瞻共识 | 营收低个位数（~2-4%） | 保守 |

质量：核心 RAN 是**成熟低增长市场**（5G 建设高峰已过，6G 尚远）。企业 5G / Network APIs 是上行期权但占比小。**产出前瞻可持续增速 ≈ 4%**（取历史下滑与前瞻 flattish 的保守中枢，企业/API 作上行期权不计入基线）。

### 护城河 — 中等（周期性侵蚀）

- 壁垒类型：全球 RAN **三寡头**（Ericsson / Nokia / Huawei，西方市场因 Huawei 被禁实为双寡头）、规模经济、运营商深度绑定、5G/6G 专利授权（IPR 现金流稳定）
- 份额趋势：西方市场因 Huawei 禁令受益，但北美（大客户 Verizon/AT&T）capex 见顶，份额稳但市场缩
- 威胁：**Open RAN**（解绑硬件、降低壁垒，长期威胁）、运营商 capex 强周期、汇率、地缘（中国市场基本失去）

## 五、安全边际 — 静态锚（周期股，用 mid-cycle 正常化 EPS）

周期股，公式用 **mid-cycle 正常化 EPS**（不用谷底也不用一次性垫高的 GAAP）。剔除 Q4'25 处置收益。

```
mid-cycle 正常化 EPS = 6.0 SEK / 9.65 = $0.62
  （TTM 正常化 5.71 SEK / 2025 正常化 6.59 SEK 的中枢；利润率恢复中）
g（前瞻，不砍半）= 4%          # RAN flattish + 历史下滑的保守中枢
合理 PE   = min(8.5 + 4, 30)  = 12.5x   # 低增长周期，PE 低
合理价    = 0.62 x 12.5        = $7.77
质量判定  = 平庸/好公司(周期) x 一般麻烦  -> 折扣系数 x0.60
满仓目标  = 7.77 x 0.60         = $4.66
```

**现价 $10.79 vs 合理价 $7.77 → 现价高于合理价 39% → 买贵了。**

敏感性（g / EPS 口径）：

| 情景 | 正常化 EPS | g | 合理 PE | 合理价 | 满仓（×0.60） |
|---|---|---|---|---|---|
| 基准（mid-cycle） | $0.62 | 4% | 12.5x | $7.77 | $4.66 |
| 乐观（margin 达标 16% + g 5%） | $0.75 | 5% | 13.5x | $10.13 | $6.08 |
| 保守（TTM 正常化 + g 3%） | $0.59 | 3% | 11.5x | $6.79 | $4.07 |

即便最乐观情景（EBITA margin 打满 16% + g 5%），合理价 $10.13 仍 ≈ 现价，**无安全边际**。

护栏检查：
- **周期股用 mid-cycle**：未用一次性垫高的 GAAP EPS（7.54 SEK），改用正常化 6.0 SEK ✓
- **屏幕 P/E 失真已剔除**：8.08B SEK 处置收益已扣，正常化 P/E 17.4x 才是真口径 ✓
- **回购不进 g**：SEK 15B 回购是安全边际来源，不抬合理 PE ✓
- **净现金托底薄**：净现金 $2.6B 仅 7% MCap，不足以做厚安全垫 ✓
- **汇率风险**：SEK 财报、USD 计价，汇率波动直接影响 ADR 价值 ⚠️
