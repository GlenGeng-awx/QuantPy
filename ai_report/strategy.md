# 组合策略 — 两 Sleeve 非对称打法

> 更新: 2026-07-04　哲学: 格雷厄姆(好价格) + 巴菲特(好公司遇暂时麻烦)
> 核心: **美股用巴菲特集中(被忽视的伟大公司 + 现金等待)，中国用格雷厄姆防御型(指数+债、25-75% 分位调、月度再平衡)**。两市场风险性质不同 → 工具不同。

---

## 为什么非对称（不是随意，是被利率差和风险性质"算"出来的）

| | 美国 | 中国 |
|---|---|---|
| 市场整体 | 贵（CAPE ~98 分位） | 便宜（中概 20-35 分位） |
| 机会性质 | **个案**（极少数错杀的伟大公司） | **系统**（整篮子均值回归） |
| 主要风险 | 估值 de-rating（系统性偏贵） | 个案暴雷（VIE / 政策 / 退市） |
| 对的工具 | **巴菲特集中**（几只 + 现金） | **格雷厄姆防御分散**（指数+债，机械再平衡） |
| 防御资产 | **现金/短债 @~4.5%**（被付钱等待） | **国债 @~1.7%**（压舱 + 降息顺风 + 再平衡弹药） |

**关键**：美股无风险利率(4.48%) > 股票盈利收益率(QQQ 前瞻 4.1%) → 现金为王、只挑最贵的错杀；中国债(1.7%)低而股便宜 → 往股 tilt、债做压舱。利率差决定了打法。

---

## Sleeve A — 美股（巴菲特集中）

**规则**：
1. **只买"伟大 + 被错杀"的极少数个股**（三✓ 或安全边际充足）。伟大公司平时不给好价格，只在暂时麻烦时才给
2. **不碰大盘指数**（VOO/QQQ/SOXX 系统性偏贵，CAPE 98 分位、风险溢价为负）
3. **现金/短债是核心仓位**：无风险 4.5% > 股票盈利收益率，**被付钱等待**，持大量现金是纪律不是踏空
4. 分批建（先 5% 再 5%），目标单只 10%；等其余伟大公司(NVDA/V/MA/MSFT/AAPL/LLY/COST)遇暂时麻烦大跌再纳入

**当前可动用清单**（据 comparison.2026-07-04）：
- **ADBE**（$219.70，可满仓，三✓ 最干净，AI 恐慌错杀，FCF yield 12%）— 唯一清晰满仓标的
- **PYPL**（$45.47，小仓，半结构瑕疵）、**NVO**（$50.43，小仓，GLP-1 落后者，Danish ADR）— 小仓/观察
- 其余美股伟大公司全部"仅贵"→ 观察清单，各自 price 文件的"等回调价"触发
- **其余大比例现金/短债 @~4.5%**

## Sleeve B — 中国（格雷厄姆防御型）

**规则**：
1. **股 + 债，权益 25%-75% 区间，按估值分位调，月度机械再平衡**
2. 股用**指数**（沪深300 510300 打底 + KWEB/中概互联网取更便宜的 tilt），**不押单只**——用篮子吃均值回归，分散掉 VIE/政策的单点暴雷（KWEB 已含腾讯/BABA/JD/PDD）
3. 债用**中国国债**（收益率低但通缩环境有降息顺风 + 压舱 + 再平衡弹药）
4. **分位→仓位映射**：`权益% = 75% − 估值分位 × 50%`（分位越低越买）
   - 分位 <25 → 权益 62-75%（便宜，加仓）
   - 分位 25-75 → 权益 37-62%（中性区）
   - 分位 >75 → 权益 25-37%（贵，减仓）

**当前配置**（据 comparison.indices.2026-07-04）：
- 沪深300 @~63 分位 → 权益 ~43%；KWEB @~28 分位 → 权益 ~61%
- 混合中国股 sleeve 加权 ~45-50 分位 → **权益 ~50% / 国债 ~50%**，月度再平衡
- **单只例外**：若要个股密度，**0700 腾讯**是唯一可直接持有的（HK 上市、非 VIE ADR、退市风险低，可满仓区）；TME/JD 是 ADR（VIE 风险），归入 KWEB 篮子处理而非单押

## Sleeve C — 宏观对冲（框架外，仓位纪律）

- **黄金**（GLD/518680）≤5-10%：对冲美股 de-rating + 人民币/地缘风险。已 $4,100 偏高，只留底仓不追
- **BTC** ≤3-5% 且**视作可归零**
- 无合理价、不摊平、不套"四毛买一块"——纯宏观卫星仓

## 再平衡纪律（月度）

1. **刷 price**：`python3 -m fundamental.cheap {标的}`（省 token），更新各 price.md 现价/分位/定档
2. **Sleeve B 机械调仓**：重算中国股 sleeve 加权估值分位 → 按 `75% − 分位×50%` 定权益目标 → 偏离 >5% 则再平衡（买跌卖涨）
3. **Sleeve A 事件驱动**：不机械调；只在(a)持仓触及"降级信号"或(b)观察清单的伟大公司跌到"等回调价"时动
4. **对冲仓**：季度看一次，仓位纪律为主，不择时

## 当前定位快照（2026-07-04）

- **整体偏防御**：美股 sleeve "少买多等"（现金 + 短债重，仅 ADBE 满仓级），中国 sleeve "机械满配区间"（股债 ~50/50）
- **权益敞口集中在**：美股极少数错杀（ADBE）+ 中概篮子（KWEB / 0700 腾讯）

## 风险与前提（必须记住）

- **中概结构性折价是真的**（地产/人口/债/VIE），不能假设必回归——防御型分散 + 债压舱 + 股息/回购正是为此，即便不涨也有缓冲，不赌单一反转
- **美股可以贵更久**（2021 到顶用了两年）→ 是"现金收息等"，不做空、不预测拐点
- **分位数据有源间差异**，月度机械再平衡本身就是纪律工具，不依赖精确点位
- 总资金 / 币种敞口 / 期限 / 风险承受度由本人定；本文件是框架不是投顾指令
