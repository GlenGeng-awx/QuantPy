## 二、估值分位（区间结构 + 同业 + de-rating）

**非周期股**（互联网平台），走正常分位——但正处"广告熄火 + AI 云接棒"范式切换，屏幕 P/E 高度失真。

| 口径 | 数值 | 来源 |
|------|------|------|
| 屏幕 P/E TTM | **负 / N.M.**（TTM 稀释 EPS −¥1.04，Q3'25 一次性 ¥16.6B 减值） | combine |
| 正常化 P/E（剔一次性） | **21.3x**（$113.3 / $5.31） | 本报告第五步 |
| 5 年平均 P/E | ~16x（min 15.19、近期 18-49x 剧烈波动） | [macrotrends](https://www.macrotrends.net/stocks/charts/BIDU/baidu/pe-ratio)、[public.com](https://public.com/stocks/bidu/pe-ratio) |

⚠️ **背离方向 = 高估贵反向**：屏幕 P/E 负/极低（Q3 ¥16.6B 一次性减值 + FY 投资损失压低 GAAP 利润）→ **正常化后反而更贵**（21x），属 TCOM/MSFT 一类"一次性损益扭曲 → 正常化更贵"。表面便宜是陷阱。

## 五、安全边际 — 静态锚（合理价 / 满仓目标）

利润被一次性麻烦压低（Q3 减值），用正常化 NI；但**注意正常化后仍不便宜**。

```
正常化 EPS = 正常化 NI ¥12.92B / 343M / 7.1 = $5.31/ADS   # 剔 ¥19.2B 一次性投资/减值损失
g（前瞻，不砍半）= 6.8%                                       # 分析师共识，AI 期权不入基线
合理 PE   = min(8.5 + 6.8, 30) = 15.3x
合理价    = 5.31 x 15.3          = $81
质量判定  = 平庸 / 麻烦存疑        -> 折扣系数 x0.60
满仓目标  = 81 x 0.60            = $49
```

敏感性（g 口径 / EPS 口径）：

| 情形 | 正常化 EPS | g | 合理 PE | 合理价 | 满仓（x0.60） |
|---|---|---|---|---|---|
| 基准（正常化核心+iQiyi，g=6.8%） | $5.31 | 6.8% | 15.3x | **$81** | $49 |
| 乐观（AI 云拉动 g=10%） | $5.31 | 10% | 18.5x | $98 | $59 |
| 保守（广告继续塌，g=2%） | $5.31 | 2% | 10.5x | $56 | $34 |

护栏检查：
- **g 质量不合格**：`FCF − SBC = −¥11.4B < 0` -> 严格说**不配成长估值**，此处 g=6.8% 已很保守，合理价 $81 或已偏高
- **回购不进 g**：$50 亿回购价值体现在缩股 + 安全边际，不抬 PE
- **净现金托底薄**：仅 8% MCap，不足以做资产锚打底
- **AI/robotaxi 作上行期权**：不计入 g，是免费看涨期权（唯一亮点）
