**结论（背离方向）**：**经营维度（①）机械看丑、实则补贴战自伤且 Q1'26 已回正 = 错杀/困境反转机会**；同时**GAAP 净利维度（②）机械看漂亮、实则被投资收益垫高 = 正常化后经营其实不赚，屏幕 19.6x 是失真**。两条背离方向相反但指向同一动作：**别信屏幕 P/E（两头失真），锚定"补贴战收尾后的恢复经营盈利"**（第五步用 ¥30B 中枢）。这与第二步"屏幕 18.7x → 正常化 6-8x"同源。存疑点在 ①：外卖是刘强东主动杀入的无护城河红海，补贴强度一次性（SAMR 叫停）但"长期低 margin 经营外卖"是半永久稀释 → 折扣取存疑档 ×0.60。

**伤口模式**：3yr 70 + TTM 20（急坠）= **困境反转候选**。伤口来源明确——外卖补贴战砸穿经营利润（OpInc FY24 39.57B → FY25 3.69B → TTM −3.04B）。主要是"战略投入 + 补贴战"伤口（部分自伤），非需求崩塌（营收仍 +1-5%、GM 稳 16%）。关键分叉：补贴战结束后 margin 能恢复多少。

## 五、安全边际 — 静态锚（利润被压低型，用恢复后经营盈利估）

**新口径链条（g 不砍半、用恢复后正常 margin）**：

```
正常化 NI = 恢复后经营盈利（补贴战收尾，外卖 margin 拖累打折）

合理 PE    = min(8.5 + 6, 30)            = 14.5x
合理价     = 3.0 × 14.5                   = $43.5  （保守 $36 / 全恢复 $58）
质量判定   = 好公司 × 麻烦存疑            → 折扣系数 ×0.60
满仓目标   = 43.5 × 0.60                  = $26
```

护栏检查：
- **利润被麻烦压低型**：用恢复后经营盈利（非低谷 −3B OpInc、非投资收益垫高的 GAAP 13.84B）✓；辅以 EV/正常化盈利 4-6x、P/S 0.2x × 正常 NM 3-4% 交叉验
