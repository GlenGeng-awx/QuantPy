# CRCL（Circle Internet Group）— 基本面 base

> **静态层**：财报/前瞻/护城河/管理层/消息面。动态见 `analysis.price.md`。
> **更新: 2026-07-12**（首次分析；财报 FY25/Q1'26 ~2026-03） 货币: USD（稳定币基础设施，新近 IPO）

**质量判定**：**框架外 / 稳定币基础设施（利率敏感型周期金融股）**（理由：USDC 稳定币发行人业务模型 = 持有 USDC 储备资产（主要是美国国债）× T-Bill yield → 营收强依赖利率；P/S 6.2x、P/B 4.8x 极端贵；TTM 净利润 −2.8%、FCF −$141M 负；但 OCC Trust Bank 牌照刚获批（Jul 10）+ Nomura 合作是监管/机构化里程碑；73% 1Y 回撤深度足够）。× 麻烦 **结构偏利率周期**（Fed 降息是稳定币业务的结构性收入压制 + 1Y 大跌是利率周期顶部回落非一次性事件） → **不适用经典价值框架公式，归入"框架外周期金融股"观察档**。

**核心锚（静态，框架外口径）**：
- 营收 TTM $2.86B（主要来自储备利息）
- **收入强挂钩 T-Bill 利率**：Fed 降息 → 稳定币业务营收下台阶（与 COIN 类周期金融）
- 净现金 ~$1.5B（Cash & STI $1.52B − 债 $14.8M ≈ $1.5B）= MCap ~8.5%（中等托底）
- 稀释股 230M（IPO 后刚增发，短期稀释冲击大）
- P/S 6.2x 隐含市场对"稳定币 AUM 长期增长 + 网络效应"给了成长溢价
- 护城河：**监管/合规先占**（USDC 第二大稳定币、市值 ~$68B vs USDT ~$150B）+ Coinbase 独家分销网络

> ⚠️ **核心问题**：稳定币业务本质是"赚取储备利息"，利率下行 = 营收下行。Fed 处于降息周期 → 未来 1-2 年营收大概率萎缩。当前 P/S 6.2x 是周期顶部利润的倍数，**与 HOOD/COIN 同为"周期金融股 + 利率敏感"类型**。

| 前瞻营收共识 | 分析师估 FY26 营收下台阶（降息预期）| 营收下滑 |

**产出 g 前瞻**：**负增长或停滞**（利率下行结构性 headwind + 营收萎缩）；USDC 份额抢夺（vs Tether）有上行期权但需要 Tether 出问题才会兑现。

### 护城河 — 合规先占 + Coinbase 独家

- **合规先占**：USDC 是受监管、储备透明的稳定币；Tether 储备不透明 → 监管压力下 Tether 用户可能迁移
- **Coinbase 独家分销**：Coinbase 持有 USDC 储备利息分成（~50%），是主要发行渠道
- **监管背书**：OCC Trust Bank 牌照（Jul 10 获批）+ Nomura 机构合作 + 华尔街背书

**护城河强度**：中等 — 监管壁垒有（OCC 牌照），但业务本质利率敏感 + 产品同质化（vs USDT/FDUSD/PYUSD）+ 分销成本（Coinbase 分成）削弱价值捕获。
