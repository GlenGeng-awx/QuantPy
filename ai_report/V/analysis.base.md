## 四、增长前瞻 / 护城河 / 管理层

- **g**：营收 +7.6%、近季 +17%（跨境交易 + 数字支付渗透）。业务 g（剔回购）≈ **10%**：支付量增长 + 现金电子化长期趋势 + 增值服务
- **护城河 — 极宽**：与 MA 构成全球卡组织双寡头，网络效应（发卡行 + 商户双边）、切换成本极高、品牌信任、规模。近乎"全球支付收费站"
- **管理层**：资本配置一流（大额回购真缩股 + 增值服务扩张）；治理稳健
- **消息面**：跨境交易强劲、增值服务加速；长期威胁——稳定币/央行数字货币/实时支付（FedNow）绕开卡网、监管交换费压力（但迄今护城河未破）

**熊**：31.5x 买 10% 增长 = 为质地付满价，无安全边际；稳定币/CBDC/实时支付长期或绕开卡网、监管压交换费；近高位无麻烦可利用。
**牛**：地表最强商业模式之一（GM 81%、NM 52%、FCF machine、真缩股）；现金电子化长坡厚雪 + 跨境 + 增值服务；双寡头定价权；护城河迄今未见实质侵蚀。

## 五、安全边际 — 静态锚

有正 EPS，用公式，g 不砍半。正常化 EPS = TTM 稀释 $11.47（干净）。

```
正常化 EPS = $11.47
g（业务，剔回购）= 10%
合理 PE   = min(8.5 + 10, 30) = 18.5x
合理价    = 11.47 × 18.5        = $212
质量判定  = 伟大 × 无麻烦        → 折扣系数 ×0.75
满仓目标  = 212 × 0.75          = $159
```

敏感性：

| g | 合理 PE | 合理价 | 满仓（×0.75） |
|---|---|---|---|
| 10%（基准） | 18.5x | $212 | $159 |
| 12%（乐观） | 20.5x | $235 | $176 |

护栏：
- **回购不进 g**：g=10% 用支付量/业务增长，回购另计 ✓
- **g 质量**：`FCF−SBC = +$20.26B`、3 年增长、SBC 仅 2.1% ✓，质地满分
- **无净现金托底**：轻净债，但商业模式 + FCF 强，非风险
- **质地只进折扣不进倍数**：伟大给 ×0.75 折扣，不给更高 PE——所以 31.5x 仍判买贵
