| 口径 | 数值 | 说明 |

金融股口径：**P/E 反读要小心**——当前信贷成本低 + 高利率 NII 高 = 盈利偏周期高位，16x P/E 是"峰值盈利下的偏高倍数"。P/B 2.46x 明确处历史高位（危机时 JPM 曾 <1x）。**周期底部买点是 P/B <1.5x 或危机**，非现在。

## 四、前瞻 / 护城河 / 管理层

- **g**：银行长周期 EPS ~5-7%，但强周期性（NIM + 信贷损失摆动）。当前高利率 + 低信贷成本 = 周期高位，正常化需下调
- **护城河 — 宽（银行里最宽）**：规模、品牌信任、全能银行网络、科技投入领先、监管护城河（SIFI）。但银行业本质同质 + 高杠杆 + 强监管，非软件式护城河
- **管理层**：Jamie Dimon（传奇 CEO，接班是关注点）；资本配置纪律强、危机中屡屡逆势扩张（收购 First Republic 等）
- **消息面**：近历史高位（1Y 回撤仅 2%）；高利率 + 强信贷环境利好，衰退/信贷周期转向是最大风险

**熊**：P/B 2.46x + 16x 峰值盈利买周期性高杠杆银行 = 无安全边际；信贷周期转向（衰退 + 拨备激增）会压 EPS；利率见顶回落压 NII；近 ATH 无麻烦可利用。
**牛**：最佳银行、ROE 16%、堡垒资本、多元化抗周期、Dimon 资本配置；高利率环境持续则盈利维持；大额回购。

## 五、安全边际 — 银行锚（P/B + 正常化 P/E），公式不适用

```
min(8.5+g, 30) 公式对银行不适用（无干净 FCF/EBITDA）。用银行锚：
正常化 EPS ≈ $19-20（剔周期低信贷成本，略降 TTM $20.90）

价值买点：P/B ~1.4x ≈ $190（需衰退/信贷危机，银行历史买点）
```

护栏：
- **金融股 P/B 口径**：现价 P/B 2.46x 历史高位 = 买贵；买点在 P/B <1.5x ✓
- **周期性盈利**：当前信贷成本低 = 峰值盈利，P/E 16x 是"顶部偏高倍数"陷阱，正常化后更贵
- **无 FCF 锚**：银行现金流表不可用，勿套 FCF yield
