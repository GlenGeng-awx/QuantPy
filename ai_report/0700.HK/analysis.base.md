## 四、增长前瞻 / 护城河 / 管理层 / 消息面

### 增长前瞻（产出第五步 g）

- **历史**：3 年营收 CAGR = (751.77/554.55)^⅓ − 1 = **10.7%**；Q4'25 +13%、全年 +14%（加速）。
- **分段（2025）**：国内游戏 +18%（Delta Force + 常青）、国际游戏 ¥77.4B 首破 $10B、广告（营销服务）高双位数（AIM+ AI 广告引擎 + 微信生态）、金融科技+企业服务 +8%、**云已达规模盈利**（AI 需求驱动）。
- **前瞻**：47/52 分析师买入/强买；一致营收增速 ~11-13%。
- **产出保守可持续 g = 12%**：历史 CAGR 10.7% 与前瞻 11-13% 取中 + 轻微 mix 顺风。护栏：回购不进 g（且回购正在缩以让路 AI capex）；margin 扩张部分接近成熟，不外推。

### 护城河 — 极宽

- **微信/WeChat 13.5 亿 MAU**——中国最强网络效应 + 超级 App + 极高切换成本；视频号广告库存**远未充分变现**（最大增量）。
- **游戏**——全球最大发行商（国内寡头 + Riot/Supercell/Epic 持股 + Delta Force 爆款）。

- **云 + AI**——Hunyuan 大模型、元宝助手，云转盈利。
- 壁垒类型：网络效应 + 切换成本 + 规模 + 牌照。份额稳固。**威胁**：字节/抖音抢广告与短视频、宏观消费弱拖累广告/支付、AI capex ROI 未证实、地缘 + VIE。
