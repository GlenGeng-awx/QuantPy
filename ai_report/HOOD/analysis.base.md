## 二、估值分位（区间结构 + 同业 + de-rating）

**周期股**（券商/加密交易平台）→ **P/E 分位会误导**，改用中周期正常化 P/E + P/S 分位 + 净现金。HOOD 2021.07 IPO，"5 年"区间短且被 2022-23 GAAP 亏损扭曲（P/E 无意义）。

### 伤口模式

Income/CF TTM 满分是**周期顶特征**，非情绪伤口。这与 ADBE（情绪伤口、基本面加速）**性质相反**——HOOD 基本面在周期回吐（Q1'26 加密 −47% 已证），"便宜"的错觉来自 P/E 未反映周期，非市场错杀。

## 五、安全边际 — 静态锚（合理价 / 满仓目标）

**周期股，用中周期正常化盈利**（不能拿 2025 加密繁荣顶峰的 TTM EPS $2.06 套公式，否则周期顶估值）。

中周期正常化：TTM EPS $2.06 含 2025 加密繁荣（Q3-Q4'25 季 EPS $0.61-0.66 峰值），Q1'26 已回落至 $0.38（年化 ~$1.52）。取跨周期中枢——2024 全年 EPS $1.56（也是好年）+ Q1'26 年化 $1.52 → **中周期正常化 EPS ≈ $1.55**（剔加密峰值贡献）。

```
中周期正常化 EPS = $1.55           # 剔 2025 加密繁荣峰值,取跨周期中枢
g（前瞻，不砍半）= 17%             # 分析师共识,含净利息/订阅稳定部分
合理 PE   = min(8.5 + 17, 30) = 25.5x
合理价    = 1.55 × 25.5           = $40（中周期口径）
质量判定  = 平庸(周期) × 麻烦存疑  → 折扣系数 ×0.60
满仓目标  = 40 × 0.60             = $24
```

护栏检查：
- **周期股公式打折**：用中周期正常化 EPS，非 TTM 峰值 ✓——否则拿加密繁荣利润套 25.5x = 双重周期顶错误

- **浮存不作锚**：现金/OCF 大量来自客户应付膨胀，随交易量反向回流，安全边际不计入 ✓
- **净现金托底 ~$5B**：公司层净现金约 $5B（MCap ~$102B 的 5%），托底薄，非价值锚 ✓
