## 二、估值分位（盈利口径失效，用 P/S + 净现金/P/B）

价格战下盈利剧烈波动且转负 → P/E 失效。属**竞争+周期性下行**，P/E 不可用。

| 口径 | 数值 | 说明 |
|------|------|------|
| P/E | 负（转亏） | 无意义 |
| P/S | 0.7x | 历史低位 |
| **EV / Sales** | **~0.13x**（EV $1.4B / Rev $15.4B） | 市场几乎不给经营业务估值 |
| **P/B** | **1.2x**（MCap 11.93B / 权益 $9.9B） | 接近账面，下行保护强 |
| 净现金 / 市值 | ~88% | 极厚安全垫 |

市场给整个经营业务的 EV 仅 ~$1.4B（一家年交付 ~38 万辆、两年前还净赚 $1.5B+ 的车企）。**便宜是真的，但成因是盈利崩塌，不是错杀**——估值判断全靠第五步资产口径 + 经营能否止血。

### 伤口模式

Income 3yr=40、TTM/5Q=10 = **结构性 + 周期性恶化**（价格战结构性 + 车型切换周期性）。区别于 ADBE 情绪伤口：LI 是真实经营恶化。但 BS 85 分 + 净现金 88% → 有清算价值保护，是"困境反转期权"而非"价值陷阱"。

## 五、安全边际 — 亏损股，净现金/清算口径（EPS 公式不适用）

min(8.5+g,30) 算不了（EPS 负），按亏损/困境股护栏用资产/清算口径：

| 锚 | 值 | 判定 |
|----|-----|------|
| **净现金** | Cash+STI 93.11B − Debt 17.74B = **¥75.37B（$10.6B）= MCap 88%** | ✓ 极厚下行保护 |
| EV | MCap 11.93B − 净现金 10.6B = **~$1.4B** | 市场几乎不给经营业务估值 |
| P/B | 11.93B / 9.9B = **1.2x** | ✓ 接近账面 |
| g 质量护栏 | `FCF − SBC < 0`（FCF −¥7.59B 烧钱） | ✗ 不配成长估值（但本就是反转期权，非成长） |

**安全垫极厚（净现金 88%、P/B 1.2x，且非受限、非发债），但经营在烧钱、毛利崩塌。** 这是教科书式"困境反转期权"：下行被净现金锁住，上行赌毛利修复。EV $1.4B 意味着——若经营止血回到哪怕微利，期权大幅 ITM。折扣系数 ×0.40~0.50 档（重麻烦/结构存疑，"四毛买一块"），但亏损用净现金锚定价而非公式。

⚠️ **但"地板"在缓慢下移，不是焊死的**：现金垫含一大块负营运资本浮存（FY23 应付 +¥37B 撑起的），随销量下滑反向回流；叠加经营烧钱，现金+短投已从峰值 ¥112.8B 掉到 ¥93.1B（5 季 −¥20B）。所以是"**FCF + 营运资本双重失血**"，销量持续降则垫子缩水快于单看亏损。期权地板真实但在动。

## 综合判定（静态两行）

| 要素 | 证据 | 判定 |
|------|------|------|
| 好公司 | 车辆毛利 19.8%→6.1%、营收降、转亏、FCF 烧钱 | ✗ |
| 一次性麻烦 | 价格战结构性 + 车型切换周期性 = 半结构，非纯一次性 | ✗ / 存疑 |

（"好价格"行 + 净现金/清算安全垫 + 最终定档见 `analysis.price.md`）

归类：**困境反转期权（结构性恶化 + 净现金保护）— 费雪不合格，不是价值买入。** 与 SNAP（无净现金的价值陷阱）本质不同——LI 有 $10.6B 净现金（88% MCap）锁住下行，是风险收益不对称的期权；但费雪（好公司）严重不合格——毛利崩、烧钱、护城河被侵蚀。**不属于"好公司遇一次性麻烦"的框架猎物**（那是 ADBE 那种），而是"结构性恶化但有清算价值保护"的特殊机会。仅适合愿承担不对称投机者：**极小仓（≤3%）博毛利修复**，非价值席位。

**升级信号（转"好公司"重看）**：连续 2 季车辆毛利回 ≥15% + FCF 转正 → 脱离"期权"按反转标的重估。
**降级信号（避）**：毛利续跌 + 现金持续烧（净现金垫快速变薄）→ 期权价值衰减，回避。
