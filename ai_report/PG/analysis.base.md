## 五、安全边际 — 静态锚（合理价 / 满仓目标）

非周期股，新口径 g 不砍半。正常化 EPS = TTM 稀释 EPS $6.84（Unusual = 0，GAAP 干净无需调整）。

```
正常化 EPS = $6.84
g（前瞻，不砍半）= 4.0%              # 有机中点 3% 与 EPS 指引 5% 间保守，业务 g（剔真缩股贡献）
合理 PE   = min(8.5 + 4.0, 30) = 12.5x
合理价    = 6.84 x 12.5          = $85.5
质量判定  = 好公司 x 一般麻烦    -> 折扣系数 x0.67
满仓目标  = 85.5 x 0.67          = $57
```

敏感性（g 口径）：

| 前瞻增速 g | 合理 PE | 合理价 | 满仓目标（x0.67） |
|---|---|---|---|
| 3.0%（有机中点，更保守） | 11.5x | $79 | $53 |
| 4.0%（基准） | 12.5x | $85.5 | $57 |
| 6.0%（EPS 指引上半区，含缩股） | 14.5x | $99 | $66 |

即便用最乐观 g=6% 合理价 $99，仍低于现价 $151 约 35%。

护栏检查：
- **回购不进 g**：g=4% 用业务增长（有机销售），真缩股 -4.7% 的 EPS 贡献不抬合理 PE，只作安全边际 + EPS 基数 ✓
- **g 质量**：`FCF - SBC = +$14.5B > 0` 且强 ✓；但增长靠涨价（量 flat）非销量扩张，g 质量偏"拧毛巾"，取保守 4% ✓
- **负营运资本浮存作锚打折**：-$10.7B 浮存随销量回流，不计入安全边际锚 ✓
- **无净现金托底**：净债 $23.1B，纯盈利估值，封顶 12.5x 后无缓冲 ✓
- **框架张力**：品牌 durability 值溢价但框架不付——同 MCD/WMT/JNJ，质量只进折扣系数 ✓
