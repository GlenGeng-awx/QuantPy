## 五、安全边际 — 静态锚（合理价 / 满仓目标）

**利润被一次性麻烦压低**（GAAP TTM NI −$165M，被 $1.2B 代金券 + 罚款砸负）→ 不能用低谷 EPS，用**恢复后正常 margin 的 EPS**。

正常化推导（剔除一次性 + Developing Offerings 主动亏损后的核心盈利能力）：
- Product Commerce 营收 ~$32B × 恢复后 net margin ~3.5%（8.4% EBITDA − D&A − 税 − 利息）≈ **正常化 NI ~$1.15B**
- Developing Offerings 主动亏损作独立期权（不计入基线盈利，也不计入负拖累）
- 正常化 EPS = 1.15B / 1.85B 股 ≈ **$0.62**
- 交叉验：P/S 0.9x ÷ 恢复后 3.5% NM = 隐含正常化 P/E ~26x（与 $0.62 一致口径）

```
正常化 EPS = $0.62                    # 恢复后正常 margin，非低谷 NI；剔一次性 + 剔 Developing 亏损
g（前瞻，不砍半）= 9%                  # 取近期 7.5% 与利润率改善折中；非 19% 历史 CAGR（已放缓）
合理 PE   = min(8.5 + 9, 30) = 17.5x
合理价    = 0.62 × 17.5             = $10.9
质量判定  = 好公司 × 麻烦一般偏存疑    → 折扣系数 ×0.60（薄利 + Taiwan 烧钱 + FCF 萎缩，从严）
满仓目标  = 10.9 × 0.60             = $6.5
```

敏感性（正常化 EPS × g 双维）：

| 正常化 EPS \ g | 8%（PE 16.5x） | 9%（PE 17.5x） | 12%（PE 20.5x） |
|---|---|---|---|
| $0.55（保守 3% NM） | $9.1 / 满 $5.5 | $9.6 / 满 $5.8 | $11.3 / 满 $6.8 |
| $0.62（基准 3.5% NM） | $10.2 / 满 $6.1 | **$10.9 / 满 $6.5** | $12.7 / 满 $7.6 |
| $0.75（乐观 4% NM + Taiwan 盈亏平衡） | $12.4 / 满 $7.4 | $13.1 / 满 $7.9 | $15.4 / 满 $9.2 |

即便最乐观（4% NM + g 12% + Taiwan 转正）合理价 ~$15.4，仍低于现价 $18.56 → 无安全边际。

护栏检查：
- **回购不进 g**：g=9% 用业务增长，首次回购规模小不抬 PE ✓
- **g 质量**：`FCF − SBC = −$181M < 0`（TTM）→ **严格说不配用成长估值**，故 EPS 用恢复后正常口径而非乐观展望，且折扣系数从严 ×0.60 ⚠️
- **利润被麻烦压低**：用恢复后正常 margin EPS $0.62，非低谷 −$0.10 ✓
- **净现金托底**：含租赁后净现金接近打平，托底薄；留存收益仍 −$4.3B，现金靠发债 + 浮存，作锚打折 ⚠️
- **薄利模式**：正常化 P/E ~30x @现价对 28.8% GM 的自营电商偏贵，非高增长可豁免（g 仅 9%，PEG >3）⚠️
