### 伤口模式

## 五、安全边际 — 静态锚（合理价 / 满仓目标）

非周期股，新口径 g 不砍半。但 **CRM 的 GAAP NI（$7.24B）远低于 FCF（$14.66B）**，差额主要是收购无形摊销（非现金）——三口径给出不同结论，都列出：

**口径一：8.5+g（GAAP 正常化 EPS）**
```
正常化 EPS = 7.24B ÷ 931M          = $7.78
g（前瞻，不砍半）= 10%
合理 PE   = min(8.5 + 10, 30)      = 18.5x
合理价    = 7.78 × 18.5            = $144
```

**口径二：FCF（更适合摊销重的 SaaS roll-up）**
```
FCF/股 = 14.66B ÷ 871M            = $16.83
78% GM、+10% 增长的 SaaS 合理 P/FCF ~15-18x → 合理价 $252-303
```

**取中（业主盈利中枢）→ 合理价 ~$200**：GAAP NI 被收购摊销压低（但持续并购，摊销部分是真实成本）；FCF 含递延收入顺风（可能高估可持续现金）。取业主盈利中枢（NI + 半摊销 ≈ $11B，18x）→ 合理价 ~$200。

```
合理价    = ~$200                  （业主盈利中枢，软/FCF 口径估算）
质量判定  = 好公司 × 存疑（AI 颠覆 seat）  → 折扣系数 ×0.60
满仓目标  = 200 × 0.60             = $120
```

敏感性（口径分歧是核心）：

| 口径 | 合理价 | 满仓目标（×0.60） |
|------|--------|--------|
| GAAP 8.5+g（g=10%） | $144 | $86 |
| 业主盈利中枢（NI + 半摊销 ≈ $11B，18x，**基准**） | ~$200 | $120 |
| FCF（P/FCF 16x） | ~$277 | $166 |

护栏检查：
- **回购不进 g**：g 用业务增长（10%），缩股价值体现在安全边际 + EPS 基数 ✓
- **g 质量**：`FCF − SBC = 14.66 − 3.55 = $11.1B > 0` ✓；真缩股（−12%/季）✓；PEG 0.8 < 2 ✓
- **口径分歧 → 安全边际标"软/FCF 口径估算"**：合理价 $200 靠业主盈利中枢，非硬 GAAP，不与硬数字并列误导 ✓
- **净债 $30B**：无净现金托底，但 FCF yield 11% 足以快速去杠杆 ✓

## 综合判定（静态两行）

| 要素 | 证据 | 判定 |
|------|------|------|
| 好公司 | GM 78%、FCF 利润率 34%、OCF/NI 1.9、真缩股 12%/季、利润率扩张；但增速 10%、GW 58%、净债 $30B | ✓（有瑕疵） |
| 一次性麻烦 | 增速降级 + AI 颠覆 seat 模式恐惧 = 半结构性；Agentforce +205% 是反向证据 | △ 存疑 |

（"好价格"行 + 安全边际% + 最终定档见 `analysis.price.md`）

归类倾向：**介于"入选"与"观察"——FCF 视角是被低估的好公司，但增长降速 + AI 范式是真实的半结构性问号。** 与 META 同档（都好公司 + 好价格边际），但画像不同：CRM FCF 更便宜（11% vs META 3.5%）、真缩股（vs META 净稀释），但增速更慢（10% vs 33%）且核心模式有 AI 颠覆问号。核心多空就一条：**Agentforce 让 CRM 用 AI 变现（好公司遇一次性麻烦）还是 AI agent 颠覆 seat 模式（价值陷阱）**——正因存疑，折扣取 ×0.60。

**降级信号（基本面，非价格）**：营收增速跌破 8% / seat 数或净留存率下滑（AI 颠覆坐实）→ 滑向价值陷阱。
**升级信号**：整体营收重加速回 ≥15%（Agentforce 兑现）+ 去杠杆 → 折扣升至 ×0.67，向"入选/最佳买点"靠。
