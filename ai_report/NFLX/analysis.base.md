## 五、安全边际 — 静态锚（合理价 / 满仓目标）

非周期股，g 不砍半。**关键：TTM GAAP EPS $3.10 含 WBD $2.8B 一次性，须剔除** → 正常化 EPS $2.58。同时给前瞻口径（FY26E EPS 已不含 WBD 一次性）交叉验。

```
【TTM 正常化口径（基准，保守）】
正常化 EPS = (13.37B − 2.8B×(1−21%)) ÷ 4.33B = 11.16B ÷ 4.33B = $2.58
g（前瞻，不砍半）= 13%
合理 PE   = min(8.5 + 13, 30) = 21.5x
合理价    = 2.58 × 21.5 = $55
质量判定  = 伟大 × 一般麻烦 → 折扣系数 ×0.75
满仓目标  = 55 × 0.75 = $41

【FY26E 前瞻口径（交叉验，利润率改善型辅证）】
FY26E EPS = 51.4B × 28.5% NM ÷ 4.3B ≈ $3.41
合理价    = 3.41 × 21.5 = $73（≈ 现价，前瞻口径基本公允）
满仓目标  = 73 × 0.75 = $55
```

敏感性（TTM 正常化 EPS $2.58）：

| 前瞻增速 g | 合理 PE | 合理价 | 满仓目标（×0.75） |
|---|---|---|---|
| 11%（指引下限） | 19.5x | $50 | $38 |
| 13%（基准，指引中值） | 21.5x | $55 | $41 |
| 15%（历史 CAGR） | 23.5x | $61 | $45 |

**两口径分叉说明**：TTM 口径 $55（现价 −40% 安全边际，明确买贵）；FY26E 前瞻口径 $73（现价 −6%，接近公允）。**取保守的 TTM 口径为主锚 $55，前瞻 $73 为上限参考**——即便用最乐观的前瞻公允价 $73，现价 $77.65 仍无价值框架要求的安全边际（满仓目标 $41-55 均低于现价）。

护栏检查：
- **回购不进 g**：g=13% 用业务增长（营收 + 广告），缩股价值体现在 EPS 基数不抬合理 PE ✓
- **g 质量**：`FCF − SBC = +$11.45B > 0` 且 3 年增长、SBC 仅 0.9% ✓；PEG ~1.5（偏高）
- **利润被一次性垫高（反向）**：WBD +$2.8B 垫高 TTM EPS → 正常化后**更贵**（30x 而非 25x）→ 屏幕 P/E 低估了贵 ✓（已剔除）
- **无净现金托底但轻净债**：净债 ~$2B，可忽略 ✓
