## 五、安全边际 — 静态锚（合理价 / 满仓目标）

非周期股，新口径 g 不砍半。**GAAP 干净，Unusual 近零 → 正常化 EPS = TTM 稀释 EPS $8.26**（无需调整）。

```
正常化 EPS = $8.26（GAAP 干净）
g（前瞻业务增长，不砍半；回购不进 g）= 7.5%
合理 PE   = min(8.5 + 7.5, 30) = 16.0x
合理价    = 8.26 × 16.0        = $132
质量判定  = 伟大 × 无麻烦（现价高位）→ 折扣系数 ×0.75
满仓目标  = 132 × 0.75          = $99
```

敏感性（g 口径）：

| 前瞻增速 g | 合理 PE | 合理价 | 满仓目标（×0.75） |
|---|---|---|---|
| 6.4%（历史 3yr CAGR，保守） | 14.9x | $123 | $92 |
| 7.5%（前瞻共识中枢，基准） | 16.0x | $132 | $99 |
| 10%（服务加速乐观） | 18.5x | $153 | $115 |

护栏检查：
- **回购不进 g**：g=7.5% 用业务增长（营收 + 服务），**不含缩股驱动的 EPS 增长**——AAPL 是典型回购抬 EPS，g 严格取业务增速，回购价值仅体现在缩股基数 + 低价回购的安全边际 ✓
- **g 质量**：`FCF − SBC = +$115.7B > 0` 且巨额 ✓；但营收 3yr CAGR 仅 6.4% → g 上限受限 ✓
- **无一次性项**：GAAP=正常化，无背离 ✓
- **高增长封顶不触发**：g=7.5% 远 < 21.5%，PE 16x ✓
- **AI 落后是尾部风险非期权**：与 ADBE AI-first ARR 上行期权相反，AAPL AI 落后是下行风险，不给估值溢价 ✓
