⚠️ Hess 并表使股数从 1.77B 增至 ~1.98B（发股对价），**每股口径需用并表后 ~1.98-2.0B 股**——摊薄后中周期 EPS 保守取 **~$9.5-10.3**。当期 TTM EPS $5.74 = 正常化的 ~56%（深低谷 + 一次性拖累）。

## 五、安全边际 — 静态锚（周期股口径：中周期正常化 + P/B + 股息，公式仅辅助）

周期股不套 min(8.5+g,30) 纯公式。主锚 = 中周期正常化盈利 + P/B 分位 + 股息率。

```
中周期正常化 EPS = $10.3             # 5 年 NI 均值 $19.6B ÷ 1.9B 股（保守取并表后 ~$9.5-10.3）
中周期合理 PE   = 12x                # 周期股不给成长溢价；质量略低于 XOM（杠杆升），给 12x
中周期合理价    = 10.3 × 12 = ~$124   # 交叉验
质量判定        = 好公司 × 一般麻烦   → 折扣系数 ×0.67
满仓目标        = 124 × 0.67 = ~$83   # 对应 P/B ~0.9x（深周期底区）+ 股息率 >8%
```

**辅助公式交叉验（min(8.5+g,30)，g=业务量增 5.5%）**：合理 PE = 8.5+5.5 = 14x → 略高于周期口径 12x（周期股保守取 12x），合理价 10.3×12 = $124 ✓。

**P/B + 股息交叉验**：
- 满仓目标 $83 → P/B = 83/92 = **0.90x**（深周期底，2020 曾到 0.78x）+ 股息率 $6.84/83 = **8.2%**（远超 5% 门槛）✓
- 现价 $169 → P/B 1.84x（中高位）+ 股息率 4.0%（未达 5%）

- **周期股不给成长溢价**：PE 封 12x（略低于 XOM 13x，因杠杆升 + Hess 消化），Guyana 量增不抬 PE ✓
