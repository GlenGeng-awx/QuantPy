### ★ SCORECARD 背离逐维度核查（机械信号 = 假设 → 原始财报证实/推翻 → 机会/陷阱）

BABA 是**机械看丑**（Income TTM 35、CF TTM 32、BS 3yr 48）的典型——但丑得**分两半**：一半是可恢复的一次性（错杀机会），一半是真实结构 downgrade（如实认，非机械误伤）。逐维度按图索骥：

| 维度 | SCORECARD 说（机械） | 原始财报证实/推翻 | 方向 → 结论 |
|------|------|------|------|
| **Income TTM 35**（OpInc −59%、EPS −18%、GM 5Q 44.9→34.5%） | 利润塌陷，机械大扣分，看着"经营恶化" | **半推翻**：营收仍 +2.7%、GM 全年 39.8% 稳、Cloud EBITA +35% 加速——塌陷源于**即时零售补贴战 ¥87B**（S&M ¥144→245B/+70%）+ AI 投入，是**主动烧钱非需求崩塌**；SAMR 6/17 立法叫停 → 可恢复 | **机械丑 vs 经营半健康 = 偏错杀（机会）**——但 ⚠️ 补贴战是真实现金流出非会计一次性，"恢复"是概率不是必然，故只算**偏一次性**不是纯情绪伤口 |
| **CF TTM 32**（FCF −50.7B、OCF −53%、Net Return −3.54B） | 造血能力崩、回购停，机械判"现金机器坏了" | **部分证实部分推翻**：FCF 转负 = 补贴战砸 OCF（76.21B/−53%）+ AI capex ¥127B（capex/OCF 167%）双杀；capex 是主动投入（Cloud 在兑现），补贴战会止血——但**两者叠加让 FCF 真流出、回购真急刹给 capex 让路** | **机械丑 = 部分真实**——投入期烧钱护栏触发（`FCF−SBC=−61.9B<0`），不配用乐观成长估值，第五步用恢复后口径 |
| **BS 3yr 48**（D/E 升、CR 1.79→1.28、发债 ¥117B） | 资产负债表转弱，机械扣分 | **证实（真 downgrade）**：⚠️ **净现金 ¥365B→¥180B→¥35B（两年崩，市值仅 2%）**——两年 ¥175B 回购 + capex 暴增 + 补贴战 + 发债 ¥117B 烧掉的，是真金白银流出**不是机械误伤**。Cash/Debt 1.22、D/E 0.25 仍可控，但"$41B 净现金堡垒"叙事已死，下行托底远薄于 PDD 56%/JD 38% | **机械丑 vs 财报也丑 = 同向 = 真 downgrade**（如实写，非错杀）——这是本次分析最硬的负面，堡垒不再 |
| **屏幕 P/E 14.9x**（不在评分但影响估值） | 看着不算特别便宜 | **推翻表象（背离反向）**：14.9x 用的是被 ¥87B 补贴战压低的低谷利润，**正常化（战争止血）后 P/E 降到 ~11-12x**——屏幕 P/E **高估了贵** | **屏幕数字偏贵 vs 正常化更便宜 = 低估贵反向**（BABA/BILI 类低谷利润，正常化后更便宜） |

**整体背离方向判定**：BABA 是**混合信号**——Income/CF 的机械丑**大半是补贴战 + capex 投入期造成的偏一次性伤口**（营收稳、GM 稳、Cloud 加速证实经营未崩，补贴战被监管叫停 → 机会方向）；但 **BS 维度的机械丑是真实的净现金崩塌（¥365B→¥35B），机械与财报同向 = 真 downgrade 不是误伤**，必须如实计入（下行托底薄）。屏幕 P/E 则是背离反向（低谷利润高估了贵，正常化后 ~11-12x）。**结论：好价格 + 偏一次性麻烦（机会）成立，但资产负债表 downgrade 是硬约束——不是干净的情绪错杀（对标不了 ADBE），催化明确（Cloud + 监管叫停补贴）但托底薄，故 ×0.67 折扣 + 仅小仓。**

## 五、安全边际 — 静态锚（利润被压低型，双口径）

利润被补贴战压低 → 用**恢复后 margin EPS** 交叉验，避免拿低谷利润低估。全程股价（ADS）口径。

```
稀释股数  = 2.40B ADS
正常化 EPS（深跌口径）= 正常化 NI 110.90B ÷ 7.1 ÷ 2.40B = $6.51
g（前瞻保守，商业稳 + 云拉动）= 8%
合理 PE = min(8.5 + 8, 30) = 16.5x
质量判定 = 好公司 × 偏一次性 → 折扣系数 ×0.67
```

| 口径 | 正常化 EPS | 合理价（16.5x） | 满仓 ×0.67 |
|------|------|------|------|
| **深跌口径**（当前被战争压低利润） | $6.51 | **$107** | **$72** |
| **恢复口径**（补贴战止血，OpInc 回 FY24-25 ¥130B 级） | ~$8.2 | **$136** | **$91** |
| 乐观（g=10% + 恢复，18.5x） | ~$8.2 | $151 | $101 |

护栏检查：
- **g 质量护栏**：FCF−SBC = −61.9B < 0（投入期烧钱）→ 按护栏**不配用乐观成长估值**，故 g 取保守 8%、且用恢复后利润而非外推高增长
- **净现金底**：仅 ¥35B（市值 2%）——**下行托底薄**（远不如 PDD 56%、JD 38%）；¥207B 股权投资是隐藏垫但不流动
- **回购不进 g**：g=8% 用业务增长；缩股历史 + $19B 授权价值体现在安全边际，不抬合理 PE ✓
- **利润被麻烦压低型**：用恢复后正常 margin 的 EPS，辅以 P/S × 正常 margin、Cloud SOTP 交叉验 ✓
