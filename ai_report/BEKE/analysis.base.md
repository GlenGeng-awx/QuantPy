⚠️ **周期股 P/E 会误导**：34x 屏幕 P/E 是底部利润压低分母，非"贵"信号，也非"底部买点"信号——需回到中周期正常化盈利。恢复口径 ~28x 仍偏高。P/S 1.3x 对薄毛利平台不算便宜。

⚠️ **背离方向 = 高估贵**（低谷利润）：屏幕 34x 因底部利润 -> 正常化/恢复后 EPS 更高、P/E 降至 ~28x（更便宜），属 BABA/BILI 一类"低谷利润 -> 正常化后更便宜"。但即便正常化仍 >合理 PE 12.3x。

## 五、安全边际 — 静态锚（合理价 / 满仓目标）

准周期股，利润被地产周期压低——用**恢复后周期中枢 margin 的 EPS**，辅以净现金托底。

```
恢复口径 NI = ~¥4.3B（周期中枢：TTM GAAP ¥3.39B 与 Q1 年化 ¥5.04B 之间取中枢，剔一次性收益）
恢复口径 EPS = 4.3B / 1.15B / 7.1 = $0.53/ADS
g（前瞻，不砍半）= 3.8%                                # 分析师共识，地产企稳不早于 2027
合理 PE   = min(8.5 + 3.8, 30) = 12.3x
合理价    = 0.53 x 12.3          = $6.5
质量判定  = 好公司（周期承压）/ 一般麻烦   -> 折扣系数 x0.67
满仓目标  = 6.5 x 0.67          = $4.3
净现金托底 = $6.1/ADS（37% MCap）           # 硬底，但仍 < 现价 $15
```

敏感性（恢复口径 EPS / g）：

| 情形 | 恢复 EPS | g | 合理 PE | 合理价 | 满仓（x0.67） |
|---|---|---|---|---|---|
| 基准（周期中枢 ¥4.3B，g=3.8%） | $0.53 | 3.8% | 12.3x | **$6.5** | $4.3 |
| 乐观（复苏至 Q1 年化 ¥5.0B，g=8%） | $0.62 | 8% | 16.5x | $10.2 | $6.8 |
| 悲观（TTM GAAP ¥3.39B，g=2%） | $0.42 | 2% | 10.5x | $4.4 | $2.9 |

护栏检查：
- **利润被周期压低**：用恢复口径 EPS（非底部 EPS），已避免低估锚失真
- **g 质量**：近期营收 −19%，家装/租赁部分可持续；主业随周期。取保守 3.8%
- **净现金托底算数**：来自留存收益（真赚），可做资产锚——$6.1/ADS 硬底
- **回购不进 g**：真缩股价值在安全边际 + EPS 基数，不抬 PE
- **即便乐观情形（合理价 $10.2）仍 < 现价 $15** -> 无论周期口径如何，现价买贵
