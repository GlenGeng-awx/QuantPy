## 二、估值分位（区间结构 + 同业 + de-rating）

### 护城河 — 极宽（双寡头）

- 壁垒类型：**与 Airbus 全球商用大飞机双头垄断**（规模 + 认证壁垒 + 数十年售后/零部件锁定 + 飞行员/机队标准化切换成本）；防务/航天第二支柱

护城河是**资本市场最宽的护城河之一**——大飞机制造进入壁垒近乎无法逾越，双寡头需求确定。这是"业务伟大"的根基。

## 五、安全边际 — 中周期正常化盈利（困境反转口径）

**周期股 + 困境反转**：当前 GAAP EPS $2.54 是一次性收益假象、正常化 −$8 是危机低谷，**两者都不能套公式**（护栏：低谷/危机利润 → 用恢复后正常 margin 的 EPS）。用**中周期正常化盈利** + FCF ramp 交叉验。

```
中周期正常化 EPS = ~$15         # 恢复后 47/月产能 + 13% 中周期 OpM(Morningstar);分析师 mid-cycle $15.13
g（穿越周期）    = 8%           # 全球 RPK +4-5% + 积压消化 + 份额,不用危机反弹高基数
合理 PE          = min(8.5 + 8, 30) = 16.5x
合理价           = 15 × 16.5    = $248
质量判定         = 业务伟大 × 资产负债表受损/恢复执行风险 → 折扣系数 ×0.60
满仓目标         = 248 × 0.60   = $149
```

**FCF 口径交叉验**（周期股辅助）：
```
2028E FCF ~$10B × 18-20x（航空稳态倍数） = $180-200B EV
减净债 ~$20B（去杠杆后） → 股权 $160-180B / ~780M 股 = $205-230/股（2028 恢复态公允）
折现回今日 + 执行风险折让 → 与 $248 合理价大致吻合,偏保守取 $220-248
```

敏感性（中周期 EPS × 合理 PE）：

| 中周期 EPS | 合理 PE（g=8%） | 合理价 | 满仓目标（×0.60） |
|---|---|---|---|
| $12（保守，OpM 未达标） | 16.5x | $198 | $119 |
| **$15（分析师中周期，基准）** | **16.5x** | **$248** | **$149** |
| $18（乐观，10B FCF 全兑现） | 16.5x | $297 | $178 |

护栏检查：
- **危机低谷 → 用恢复后正常 margin EPS** ✓（不套用一次性收益 $2.54 或低谷 −$8）
- **回购不进 g**：BA 零回购，g=8% 纯业务/交付增长 ✓
- **g 质量**：当前 `FCF<0` 是危机低谷，但**趋势明确转正**（OCF +135%、FCF 指引 +$1-3B），恢复态 FCF−SBC 大幅为正（SBC 仅 0.5%）✓；用恢复态而非当前
- **资产托底**：$695B 积压订单 + 现金 $21B = 需求端硬托底，下行有底（非 SNAP 式无托底）✓；但净债 $26B + 权益薄 = 资产负债表受损，故折扣取偏保守 ×0.60
- **周期股中周期口径**：EPS 用跨周期中枢 $15，非峰值/低谷 ✓
