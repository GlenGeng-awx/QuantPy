- **营收强增**：Q1'26 $3.17B、+34.3% YoY（GMV $101B +35%），加速中，未放缓
- **OpInc 真实改善**：OpM 从 11.8%→15.7%（Q4'25 达 20.3%），经营杠杆显现
- **GAAP NI 被股权 MTM 彻底扭曲**：Q1'25 −$682M、Q2'25 +$906M、Q1'26 −$581M——单看 GAAP NI 毫无意义，**必须用正常化 NI**（剔 Gain On Sale Of Security）。正常化 NI 平稳增长 283M→505M
- **GM ~48%（略降）**：从 49.8%→48% 因 Merchant Solutions（支付，低毛利）占比升至 72%，是模式演进非定价权恶化，但拉低整体 GM
- **股数几乎不缩**：dil shares 1.30-1.31B，基本持平（SBC 被抵消，非真缩股）

### 利润增长质量（增长真实，非拧毛巾）

- OpInc 74M→1.30B→1.89B（3yr）由**营收增长（+30% CAGR）驱动**，非砍费用——"接新水"型，可持续
- OpM 从极低位（1% → 17%）爬升，绝对 OpM 仍在成熟中，改善空间尚存
- 营收引擎强劲（+34%），非"利润表满分掩盖营收熄火"

## 五、安全边际 — 静态锚（合理价 / 满仓目标）

**利润被 MTM 扰动型 + 高增长封顶**——用正常化 NI（剔股权 MTM），g≥22% 一律 30x 封顶。

```
正常化 EPS = 正常化 NI $1.95B / 稀释股数 1.31B = $1.49

g（前瞻，不砍半）= ~24%（共识 22-25%，>=22% 触顶）
合理 PE   = min(8.5 + 24, 30) = 30x（硬封顶）
合理价    = 1.49 x 30          = $44.7
质量判定  = 好公司 x 无一次性困境（纯贵）-> 折扣系数 x0.67
满仓目标  = 44.7 x 0.67        = $30
```

敏感性（g 口径，均触 30x 封顶，合理价对 g 不敏感——这是"买贵了"的本质）：

| 前瞻增速 g | 合理 PE | 合理价 | 满仓目标（×0.67） |
|---|---|---|---|
| 22%（共识下限） | 30x（封顶） | $44.7 | $30 |
| 24%（基准） | 30x（封顶） | $44.7 | $30 |
| 30%（乐观，仍触顶） | 30x（封顶） | $44.7 | $30 |

> g≥22% 后合理价恒为 $44.7（30x 封顶）——**再高的增长也不能把 $119 现价变便宜**。价值框架不追高增长的核心体现。

交叉验（P/S × 正常 margin）：营收 TTM $12.37B，成熟期正常 NM ~20%（现 10.8%，OpM 在爬）→ 正常化 NI ~$2.47B → 30x = $74B 合理市值 ÷ 1.31B = ~$57/股（乐观口径，假设 margin 翻倍）。即便给最乐观的成熟 margin，合理价 ~$57 仍远低于现价 $119。

护栏检查：
- **高增长封顶**：g=24%≥22% 一律 30x，给不出安全边际 → 判"买贵了" ✓
- **回购不进 g**：SHOP 本就几乎零回购，g 全是业务增长 ✓
- **g 质量**：`FCF − SBC = +$1.65B > 0` 且 3 年增长 ✓（成长估值合格，但估值太贵）
- **利润率改善型**：已用正常化 NI + P/S×正常 margin 交叉验，避免拿低谷 EPS 高估或低估 ✓
- **现金来源打折**：净现金 5.6B 大半来自历史发股，作锚打折 ✓
