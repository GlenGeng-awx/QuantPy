# GLD（SPDR Gold Shares）— base

> **更新: 2026-07-04　类别: 非生产性资产（框架外）　货币: USD**

⚠️ **价值框架不适用**——无现金流、无盈利、无内在价值锚，**不产出合理价**。格雷厄姆-费雪-巴菲特三要素（好价格/好公司/一次性麻烦）全部无法套用：黄金不赚钱、不分红、不缩股，没有「正常化 EPS」可折现。此处仅作**宏观对冲/配置仓**，非价值席位。不套「四毛买一块」。

---

## 一、这是什么

实物黄金信托：每份 GLD 约对应 **1/10 盎司黄金**（扣年费 ~0.40% 后逐年微降，现约 0.093 oz/份）。信托只持金条、被动跟踪伦敦金现价，1:1 实物背书。买 GLD = 买黄金本身，不是买一家公司。

## 二、为什么框架失效

巴菲特对**非生产性资产**的批评：一块金子放一百年还是那块金子，不产出任何现金流、不孳息、不复利。它的价格完全取决于「下一个人愿意付多少」（greater fool），而非资产自身创造的价值。DCF 分母有了、分子恒为 0 → 内在价值公式直接失效。所以本框架**正确地拒绝给它定价**：没有合理价，只有市场价与宏观驱动。

## 三、能用的（非本框架）宏观口径

不是估值，是「判断顺风还是逆风」的参照，**不构成买卖信号**：

- **实际利率**（最强驱动）：黄金无息，与 10Y 实际利率（TIPS）反向。实际利率高 → 持金机会成本大 → 逆风；实际利率降/转负 → 顺风。当前 10Y 实际利率 ~1.89%（偏高，本应压制金价，但被下条抵消）。
- **央行购金 / 去美元化**（当前主线）：央行 Q1'26 购金 244 吨（QoQ +17%，高于五年均值），2022 以来结构性去美元化持续。这是「对利率不敏感」的政策性买盘，是当前金价强的核心。
- **避险 / 货币贬值对冲**：地缘冲突、财政赤字货币化、法币信用弱化时的储值需求。
- **美元指数**：金以美元计价，美元弱 → 金价名义抬升。

## 四、配置定位

- **角色**：宏观对冲仓 / 卫星仓。对冲的是「实际利率下行 + 法币信用弱化 + 尾部避险」，与股票组合低相关甚至负相关。
- **仓位纪律**：作为组合对冲，贵金属合计（GLD+SLV+518680）建议 **≤5–10% 组合**，单一 GLD 更小。不因「便宜」加仓（无便宜可言）、不因「看多」满仓——它是保险，不是价值标的。
- **禁忌**：不套折扣系数、不算安全边际、不设「合理价」目标。定投/再平衡优于择时。
- **与 SLV/518680 关系**：GLD/SLV 看金银比（当前 ~67，历史 50–80 区间），518680 是人民币计价的同一底层（上海金，元/克），三者本质是同一宏观押注的不同币种/金属切片，勿重复超配。
