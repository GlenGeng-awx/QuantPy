## 五、安全边际（现价代入静态锚）

```
合理价（base）  = $22
现价            = $24.30
安全边际        = 1 − 24.30 / 22 = −10%（略贵于公允）
满仓目标（base）= $11（×0.50 平庸/结构存疑 × 结构性悬崖赌反转）
现价 $24.30 vs 合理价 $22 → 高出 +10%，无安全边际
```

| 价位 | 相对现价 | 含义 |
|---|---|---|
| 满仓目标 $11 | −55% | 深度折让区（赌反转才建满） |
| 合理价 $22 | −9% | 公允，现价略高于此 |
| 现价 $24.30 | — | 6.7% 股息率支撑，但 FCF payout 108% |
| FCF yield @ $24.30 | 9.48B / 138.61B = **6.8%** | 与股息率相当 → **FCF 几乎全额派息，无缓冲**（价值陷阱特征） |

⚠️ **FCF yield 6.8% ≈ 股息率 6.7%**：自由现金流基本全额用于分红，无留存再投资/缓冲。这不是"便宜的好公司"，是"现金流勉强撑住高股息的衰退公司"。

**操作**
- 不入价值席位。现价 $24.30 略贵于公允 $22，且结构性衰退
- 若博反转（≤小仓投机，非 10% 席位）：等 **$16 以下**（牛市满仓目标）才有不对称性，且需盯 ① FCF 重新覆盖分红 ② Metsera 肥胖/双抗读出 ③ 成本削减 $7.2B 兑现
- 收息盘警示：6.7% 股息率 FCF payout 108%，分红可持续性存疑 —— 高息是补偿结构性风险，非白捡
- 止损/降级：分红削减信号 / Goodwill 减值 / 管线试验再失败 → 确认陷阱，离场
