## 一、价格粗筛结论（详表见 price）

现价 ~$24.3，距 52W 低点 +10%、EV/EBITDA 7.5x —— **确实入池**（2/6 满足）。这是"因暂时困境下跌"的合格候选，值得深查。但下跌成因是**结构性**（COVID 退潮 + 专利悬崖），非纯情绪 —— 入池后精筛见分晓。

**非周期股**，但盈利被 COVID 泡沫 + Seagen 摊销/减值严重扰动，P/E 分位失真，须用正常化。

### 伤口模式

Income/CF 3yr 中等 + 无一次性巨额单项污染 = **不是困境反转候选（无"3yr 高 + TTM 低"）**，而是**COVID 泡沫退潮后的结构性回归 + 专利悬崖叠加**。GM 从 COVID 后 58% 恢复到 74% 是好信号，但营收持续下滑 + 分红不覆盖 FCF 是结构裂痕。**结构性伤口，非情绪伤口。**

## 五、安全边际 — 静态锚（合理价 / 满仓目标）

**关键：g 为负，正常化 EPS 须扣分红不覆盖 + Seagen 摊销真实性。** adj EPS 指引 $2.80-3.00，但含成本削减一次性台阶 + 加回 Seagen 摊销（半重复性），保守取 **$2.60**。

```
正常化 EPS = $2.60             # adj $2.90 中值 × 打折（分红不覆盖 FCF + 摊销半重复不全加回）
g（前瞻，不砍半）= −2%          # 营收结构性下滑，adj EPS 靠成本削减非增长
合理 PE   = min(8.5 + (-2), 30) = 8.5x（负 g 触发封底 8.5x）
合理价    = 2.60 × 8.5         = $22
质量判定  = 平庸/结构存疑 × 结构性悬崖（赌反转）→ 折扣系数 ×0.50
满仓目标  = 22 × 0.50          = $11
```

| 熊（衰退加深，减值） | $2.20 | 8.5x | $19 | $9 |

护栏检查：
- **回购不进 g**：无回购，不涉及 ✓
- **g 质量**：`FCF − SBC = +$8.58B > 0` ✓，但 **FCF 不覆盖分红（payout 108%）** → g 质量重大瑕疵 ✗；负增长不配成长估值 → 封底 8.5x ✓
- **麻烦定性**：COVID 退潮（已完成，一次性）叠加 **Eliquis/Ibrance 专利悬崖（结构性）+ Seagen 试验失败（结构性）**→ 主要麻烦结构性 ✗
- **资产托底**：**无净现金**（净债 $50.7B）+ 有形资产被 60% 无形占据 → 无托底，若恶化加剧滑向价值陷阱 ⚠️
