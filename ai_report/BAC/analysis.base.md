# BAC（美国银行 Bank of America）— 基本面 base

> **静态层**：财报/前瞻/护城河/管理层驱动，季度或重大事件更新。动态见 `analysis.price.md`。
> **更新: 2026-07-04**（财报 FY26 Q1，2026-04-15 发）　货币: USD　**金融股口径**（P/B 替代 EV/EBITDA；FCF/现金流公式不适用）

**质量判定**：**好公司（大型全能银行，但周期+杠杆+ROE 偏低）**——第二大美国银行、全国零售存款特许经营（低成本存款护城河）、多元化 NII + 财富 + 投行；但 ROE 仅 **10.6%**（低于 JPM 16%、行业优等生 15%+），资产收益率平庸，本质高杠杆 + 盈利随信贷/利率周期摆动，非费雪式宽护城河。用 **P/B + 正常化 P/E + ROE** 锚，非 min(8.5+g,30) 公式（银行无干净 FCF；OCF −8.8B/+56.6B 剧烈摆动是放贷/存款流动，非经营）。× 麻烦 **无**（近 52 周高位，1Y 回撤仅 3%，峰值盈利）

**核心锚（静态）**：正常化 EPS ~**$3.90** → 银行合理 PE **~11.5x**（对 ROE 10.6% 的中游银行）→ **合理价 ~$45**；P/B 交叉验 合理 ~1.15x × 账面 $38.66/股 = **$44**。价值买点 P/B ~0.9x ≈ **$35**（需衰退/信贷危机，BAC 历史危机曾 <0.5x）

## 二、估值 — 金融股用 P/B，非 P/E 贵贱

| 口径 | 数值 | 说明 |
|------|------|------|
| P/E TTM | ~14.8x（P=$59.67 / E=$4.03） | 银行历史 10-13x，处中枢上方；低信贷成本下是"峰值盈利偏高倍数" |
| **P/B** | **~1.54x**（$59.67 / 账面 $38.66） | 银行核心估值；**10 年中位 1.14x**，当前 35% 上方 |
| P/TBV | ~1.79x（TBV/股 $28.84） | GW $69B 大（并购累积），P/TBV 明显高于 P/B |
| ROE | **10.6%**（Q1 2026） | 中游——低于 JPM 16%、行业优等；决定合理 P/B 只配 ~1.1-1.2x |

**自身 P/B 5 年区间**：约 0.85x（2023 银行恐慌/AOCI 债券浮亏低点）~ 1.55x（2026 近高位），中位 ~1.14x（[MacroTrends](https://www.macrotrends.net/stocks/charts/BAC/bank-of-america/price-book)、[GuruFocus](https://www.gurufocus.com/term/pb-ratio/BAC)）。当前 1.54x 处 5 年 **~85th+ 分位**（高位）。

**同业对比（P/B / ROE）**：

| 银行 | P/B | ROE | 备注 |
|------|-----|-----|------|
| **BAC** | **~1.54x** | **10.6%** | ROE 偏低，P/B 与之匹配 |
| JPM | ~2.46x | 16.2% | 最强，溢价合理 |
| WFC | ~1.4x | ~12% | 相近 |
| C（花旗） | ~0.8x | ~8% | 更差 ROE，更低 P/B |

BAC 的 1.54x P/B 表面比 JPM 2.46x 便宜，但**低 P/B 匹配低 ROE**（10.6% vs 16%）——不是错杀，是资产盈利能力真实较弱。金融股"P/B 便宜"须配 ROE 一起看，否则是价值陷阱信号。

⚠️ **AOCI 债券浮亏 overhang**：BAC HTM 组合（长久期低息债）在高利率下含大额未实现浮亏，压低 TBV（TBV/股 $28.84 vs 账面 $38.66，差额部分即累积 OCI 亏损 −$11B）。利率下行则浮亏收窄、TBV 修复（上行期权），但账面质量当前受此压制。

### 现金流 — 银行口径不适用

- OCF 三年 44.98B→−8.8B→12.61B 剧烈摆动，是资产负债表流动（放贷/存款/证券买卖），**非经营现金流意义**——银行忽略 FCF/OCF
- 股东回报：TTM 回购 **$25.15B** + 分红（Q1 回购 $7.2B + 分红 $2B），力度大、真缩股 ✓
- SBC/Rev 3.5%，温和

**g 质量**：EPS 增长含大量缩股 + NII 周期性修复——**业务 g 取 5-7%**（银行长周期中枢）。当前低信贷成本 = 周期高位，衰退时拨备激增会压 EPS，正常化需下调。回购不进 g。

### 护城河 — 中宽（银行里第二档）

- 壁垒类型：全国零售分行 + 数字银行网络、**低成本核心存款特许经营**（沉淀活期存款 = 结构性低资金成本护城河）、财富管理（Merrill）、规模、SIFI 监管牌照
- 份额趋势：美国零售存款前二稳固；但 ROE 10.6% 说明资产端盈利能力不及 JPM，护城河"宽但不深"
- 威胁：净息差随利率周期摆动、金融科技存款竞争、信贷周期转向

### 管理层

- **CEO Brian Moynihan**（2010 至今，15 年长任期）：危机后重建 BAC 的功臣，资本配置稳健（大额回购 + 提分红），"负责任增长"策略；接班计划是关注点
- 资本配置：TTM 回购 $25.15B（真缩股 −4.5%/5Q）+ 稳定分红——股东回报力度大 ✓
- 说到做到：上调 NII 指引 +6-8%，Q1 EPS beat

### 消息面

- Q1 2026 EPS $1.11 beat，NII 修复，Moynihan 称"消费者在花钱、信贷质量非常好且在改善"（[CNBC](https://www.cnbc.com/2026/04/15/bank-of-america-bac-earnings-q1-2026.html)）
- 全年 NII 指引上调至 +6-8%
- 股价近 52 周高位（1Y 回撤仅 3%），无困境

### 熊 / 牛逻辑

**熊**：P/B 1.33x（70th 分位、10 年中位 1.14x 上方）+ 12.8x 峰值盈利买 ROE 仅 10.6% 的中游银行 = 无安全边际；低 P/B 匹配低 ROE，非错杀；信贷周期转向（衰退 + 拨备激增）压 EPS；AOCI 债券浮亏压 TBV；近 ATH 无麻烦可利用。

**牛**：低成本存款特许经营 + NII 修复（低息资产重定价 +6-8%）+ 大额回购缩股 4.5%/5Q + 街一致偏多（目标 $63）；利率下行则 AOCI 浮亏收窄、TBV 修复（上行期权）；Moynihan 资本配置稳健。

## 五、安全边际 — 银行锚（P/B + 正常化 P/E），公式不适用

```
min(8.5+g, 30) 公式对银行不适用（无干净 FCF/EBITDA）。用银行锚：
正常化 EPS ≈ $3.90（TTM $4.03 略降，剔周期低信贷成本）
银行合理 PE ≈ 11.5x（ROE 仅 10.6% 的中游银行，给行业中枢略下）
合理价 ≈ 3.90 × 11.5 = ~$45
P/B 交叉验：合理 P/B ~1.15x（10 年中位 1.14）× 账面 $38.66/股 = $44 ✓（两法吻合 ~$44-45）
价值买点：P/B ~0.9x ≈ $35（需衰退/信贷危机；BAC 历史危机曾 <0.5x）
```

护栏：
- **金融股 P/B 口径**：现价 P/B 1.33x 处 70th 分位、10 年中位上方 = 略贵；买点在 P/B <1.0x ✓
- **低 P/B 匹配低 ROE**：1.33x 表面比 JPM 便宜，但 ROE 10.6% << JPM 16%，非错杀——金融股 P/B 须配 ROE ✓
- **周期性盈利**：当前信贷成本低 = 峰值盈利，12.8x 是"顶部偏高倍数"陷阱，正常化后更贵 ✓
- **AOCI 浮亏**：TBV/股 $28.84 << 账面 $38.66，账面质量受债券浮亏压制，做锚打折 ✓
- **无 FCF 锚**：银行现金流表不可用，勿套 FCF yield ✓
