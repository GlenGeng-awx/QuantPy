## 三、财务健康 — 放贷模式，SCORECARD 机械口径需重解

SCORECARD 九宫格（USD）：Income 3yr=28 / TTM=90（5Q 数据不足）；CF 3yr=12 / TTM=15（放贷 receivables 扩张压垮 OCF/FCF，机械低分）；BS 3yr=48 / 5Q=100。

### 利润表 — 逐季（IPO 后季度有限）


- **营收强加速**：Q1'26 $1.01B、**+44% YoY**（vs Q1'25 $701M），美国 +67%，GMV $33.7B +33%（[Klarna IR 2026-05-14](https://investors.klarna.com/News--Events/news/news-details/2026/Klarna-Delivers-Strong-Start-to-2026-With-1Bn-Revenue-and-68M-Adj--Operating-Profit/default.aspx)）

- **利息费用吞噬利润**：Q1'26 IntExp $155M = OpInc $188M 的 82%——放贷机构本质，利息覆盖 TTM 仅 **0.8x**（OpInc 587M / IntExp 704M），杠杆敏感

| 项目 | TTM | 说明 |

### 伤口模式

**非"伤口反转"——是"IPO 后首次证明盈利可持续"的早期阶段**。3yr Income=28（历史巨亏）→ TTM=90（刚转正）是"从亏损爬向微利"。区分：非情绪伤口（无优质基本面被错杀），非结构性恶化（业务在增长转正），而是**未经检验的早期盈利 + 信贷周期风险**。

## 五、安全边际 — 框架外，锚为估算（非公式硬值）

**公式不适用**：TTM GAAP 亏损、Q1'26 刚打平，无稳定正常化 EPS 套 `EPS × min(8.5+g,30)`。属"结构性微利 + 早期放贷机构"框架外，用前瞻正常化盈利能力 + P/S 交叉验，质量差/麻烦存疑 → 折扣取重档（×0.40~0.50 赌反转）。

```
前瞻正常化盈利能力（FY26 指引外推）：
  rev（FY26）        ≈ >$4.3B（GMV $155B × 2.8%）
  adj op income      ≈ 6.9% × 4.3B = ~$300M
  利息/税后正常化 NI  ~$150-250M（利息费用高、杠杆敏感 → 保守）
  正常化 EPS         ≈ $0.40（保守）~ $0.66（乐观）/ 378M ADS

估值锚（两法交叉）：
  法A 前瞻 PE：成长型放贷机构给 20-25x（g~26% 但质量差/无护城河，不给 30x）
    合理价 ≈ $0.40×22 ~ $0.66×22 = $8.8 ~ $14.5
  法B P/S：BNPL 微利商给 1.5-2.0x（对齐同业、不给成长溢价）
    合理价 ≈ 1.5×($4.3B/378M) ~ 2.0× = $11.4 ~ $22.8（P/S 法宽且乐观）
  取交集偏保守 → 合理价锚 ~$10-15
质量判定  = 平庸 / 麻烦存疑    → 折扣系数 ×0.40~0.50（赌反转）
满仓目标（若入池）≈ 合理价锚 ~$11 × 0.45 ≈ $5（远低于现价，实际=不出手）
```

敏感性（估值极度依赖"盈利能否稳定爬坡"）：

| 情景 | 正常化 EPS | 倍数 | 合理价 |
|---|---|---|---|
| 盈利爬坡兑现（牛） | $0.66 | 25x | ~$16.5 |
| 基准（微利，谨慎） | $0.50 | 20x | ~$10 |
| 信贷周期恶化（熊） | $0.20 或再亏 | — | <$5 或无锚 |

护栏检查：
- **g 质量不合格**：`FCF−SBC` 深负、增长靠发债/发股放贷 → 不配用成长估值公式，走框架外 ✓
- **高增长不给 30x**：g~26% 但护城河弱 + 信贷风险 → 只给 20-25x ✓
- **现金锚打折**：$4B 现金来自发债+IPO 发股（非留存），$13B deferred 是浮存 → 作托底大幅打折 ✓
- **杠杆/利息敏感**：利息覆盖 0.8x，融资成本上行是核心逆风 ✓
