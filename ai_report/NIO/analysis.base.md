## 二、估值分位（盈利口径失效，用 P/S + 净现金/P/B）

长期巨亏、盈利刚触平 → P/E 失效。属**长期烧钱模式 + 价格战**，P/E 不可用，用 P/S + 净现金/P/B。

| 口径 | 数值 | 说明 |
|------|------|------|
| P/E TTM | 负（TTM −¥3.76） | 无意义；Q4'25 曾单季非 GAAP +$104M |
| P/S | 0.8x（MCap $12.00B / Rev ¥100.99B=$14.2B） | 低位，但营收含低毛利走量 |
| EV / Sales | ~0.8x（净现金几乎为零，EV≈MCap） | 市场给经营业务估值≈全部市值——因无净现金可扣 |
| **P/B** | **~1.9x**（MCap $12.00B / 权益 ¥4.16B=$0.6B） | ⚠️ 权益近乎清零，P/B 高且不稳（H1'25 曾负） |
| 净现金 / 市值 | **~6%（仅 ¥5B 非受限净现金）** | ⚠️ 三家最薄，几乎无下行保护 |

⚠️ **de-rating 判断**：NIO P/S 0.8x 看似便宜，但**权益近乎清零（$0.6B）+ 净现金几乎为零** → 便宜的成因不是错杀，而是**资产负债表风险 + 长期不盈利被市场惩罚**。P/B ~1.9x 反而不低（因分母权益被亏损蚕食殆尽）。这是"便宜的烂资产负债表"，非"错杀的好公司"。

## 五、安全边际 — 亏损股，净现金/清算口径（EPS 公式不适用）

min(8.5+g,30) 算不了（TTM EPS 负），按亏损/困境股护栏用净现金/清算口径 + 前瞻盈利：

| 锚 | 值 | 判定 |
|----|-----|------|
| **净现金（仅非受限）** | Cash+STI ¥31.77B − 债 ¥26.57B = **~¥5.2B（$0.7B）= MCap ~6%** | ⚠️⚠️ 极薄，几乎无托底（LI 88%、XPEV 18-31%） |
| 含受限现金口径 | 官方总流动性 ¥48.2B，但含受限 34B（电池资产沉淀）+ 需扣债 | 名义大，自由现金少 |
| EV / Sales | EV≈MCap（净现金≈0）/ Rev $14.2B = **~0.8x** | 市场给经营业务估值≈全部市值 |
| P/B | ~1.9x（权益仅 $0.6B，H1'25 曾负） | ⚠️ 分母被亏损蚀空，不稳 |
| **前瞻盈利锚** | FY26 分析师仍预期微亏（−¥1.69 EPS）；管理层目标 Q4 单季盈利 | 盈利兑现最不确定 |
| g 质量护栏 | `FCF − SBC`：FCF −3.07B − SBC ~1.7B = **−4.8B < 0**（烧钱 + SBC 伪 FCF） | ✗✗ 严重不配成长估值 |

**安全垫几乎不存在（净现金 6%、权益 $0.6B、慢性稀释）**，经营在改善但资产负债表随时可能需再融资。这不是"净现金锁底的期权"（那是 LI），而是**"托底极薄 + 持续稀释的脆弱反转"，本质接近价值陷阱**。EV≈全市值意味着市场已把 NIO 当"活着的经营业务"定价，不给清算折价保护——若盈利兑现不及 + 再融资摊薄，下行无净现金锁死。折扣系数概念上 ×0.40（重麻烦/结构存疑）但**亏损用清算锚，而 NIO 清算锚几乎为零**。

⚠️ **无净现金托底 → 滑向价值陷阱一侧**：按 CLAUDE.md「结构性恶化」分类——LI 有净现金/清算价值（≈市值）= 困境反转期权；**NIO 净现金几乎为零 + 权益近清零 + 慢性稀释 = 更接近"无托底的价值陷阱"**。仅靠盈利兑现的希望支撑，下行保护最弱。

**降级信号（避）**：Q4'26 盈利落空 + 再次大额增发稀释 + GM 回落 + 现金消耗加速 → 价值陷阱坐实，回避。
