## 二、估值分位（区间结构 + 同业 + de-rating）

**类周期股（信用周期）**——BNPL 放贷盈利随信用周期与利率剧烈摆动，且 AFRM 刚脱离多年亏损，**P/E 分位无意义**（2022-25 长期亏损，历史 P/E 为负/无穷）。改用 P/B + P/S + 前瞻 P/E + 信用指标。

### AFRM 自身估值区间

| 口径 | 数值 | 来源 |
|------|------|------|
| P/E TTM | **76.9x**（E=$1.10，刚转正、基数极低） | cheap 引擎 |
| P/S | **7.1x**（营收 $3.97B） | cheap 引擎 |
| **P/B** | **~7.5x**（BV/share ~$11.3，Sep'25 官方 $10.00） | [GuruFocus BVPS](https://www.gurufocus.com/term/Book+Value+Per+Share/AFRM/Book-Value-per-Share/Affirm-Holdings) |
| 前瞻 P/E | 极高（FY26 EPS 仍薄） | — |

**背离方向**：AFRM 属**低基数利润 → 高估**方向（刚转正，EPS $1.10 从极低基数放大 P/E 到 77x）。但即便正常化/前瞻 EPS 翻倍到 ~$2.2，$84.36 仍 30x+ 封顶，**无安全边际**。

⚠️ **de-rating 陷阱反向**：AFRM 股价从 52 周低点 $42.5 反弹 **+99%**（近翻倍），处近 52 周高点。这是**re-rating 泡沫**（GAAP 转正 + AI 叙事驱动），不是错杀低位。

| 公司 | 口径 | 备注 |

## 五、安全边际 — 静态锚（合理价 / 满仓目标）

**公式勉强适用**（刚有正 EPS），但需重度打折：正常化 EPS = TTM GAAP EPS **$1.10**（无一次性项，但基数极低、盈利脆弱、含放贷杠杆风险）。

```
正常化 EPS = $1.10                       # 刚转正，无 Unusual
g（GMV/营收前瞻）= ≥30%                   # 但价值框架 g≥22% 封顶
合理 PE   = min(8.5 + 30, 30) = 30x（封顶）  # 不追高增长
合理价    = 1.10 × 30          = $33
质量判定  = 平庸/存疑（杠杆放贷、信用周期、净稀释）× 麻烦不适用（高估非困境）→ 折扣系数 ×0.60
满仓目标  = 33 × 0.60          = $20
```

敏感性（正常化 EPS 假设，g 全部封顶 30x）：

| 正常化/前瞻 EPS 假设 | 合理 PE | 合理价 | 现价 $84.36 安全边际 |
|---|---|---|---|
| $1.10（TTM，基准） | 30x | $33 | **−156%（严重高估）** |
| $1.65（乐观，+50%） | 30x | $50 | −70% |
| $2.20（极乐观，翻倍） | 30x | $66 | −28% |

**即便盈利翻倍到 $2.20，$84.36 仍无安全边际（−28%）。** 市场按 P/S 7x / P/B 7.5x 的"平台"逻辑定价，价值框架按放贷机构盈利根本够不着。

护栏检查：
- **高增长封顶**：g≥30% 一律按 30x，价值框架不追高增长；给不出安全边际 → 判"买贵了" ✓
- **g 质量**：`FCF − SBC = +$484M`（勉强正），但**零回购净稀释 = 重大负面** ⚠️；放贷 FCF 口径不可靠
- **放贷机构**：应叠加 P/B 透镜（7.5x 泡沫）+ 信用损失（拨备 6%、NCO 3.5% GMV、衰退敏感）；公式高估其真实价值
- **周期/信用风险**：盈利刚转正、信用周期未经完整衰退检验，正常化 EPS 可能高估穿越周期盈利能力 ⚠️
