# SLV（iShares Silver Trust）— base

> **更新: 2026-07-04　类别: 非生产性资产（框架外）　货币: USD**

⚠️ **价值框架不适用**——无现金流、无盈利、无内在价值锚，**不产出合理价**。白银不赚钱、不分红、不缩股，没有「正常化 EPS」可折现。仅作**宏观对冲/配置仓**，非价值席位。不套「四毛买一块」。

---

## 一、这是什么

实物白银信托：每份 SLV 约对应 **1 盎司白银**（扣年费 ~0.50% 后逐年微降）。信托只持银条、被动跟踪伦敦银现价，实物背书。买 SLV = 买白银本身，不是买公司。

## 二、为什么框架失效

同 GLD 的巴菲特「非生产性资产」批评：白银搁那儿不产出任何现金流，价值只等于「别人愿付多少」。DCF 分子恒为 0，内在价值公式失效，框架**正确地拒绝定价**。白银比黄金还多一层波动来源（工业需求），价格更暴、更难锚，反而更不适合价值口径。

## 三、能用的（非本框架）宏观口径

**非买卖信号**：

- **实际利率**：同金，与 10Y 实际利率反向（无息资产机会成本）。当前 ~1.89%（逆风）。
- **金银比**（银特有）：当前 ~67（7/2 收 66.9），历史自由市场区间约 50–80。比值高 = 银相对金偏便宜（银有补涨空间）；比值低 = 银相对金偏贵。当前 67 属中性偏中枢，不极端。
- **工业需求**（银特有，占需求 ~50%+）：光伏、电子、EV。工业属性令银在「货币催化 + 工业景气」双击时弹性大于金（7/2 银 +3.85% vs 金 +2.5%），但衰退时也跌得更狠——「涨时更疯、跌时更惨」。
- **避险 / 去美元化**：跟随金的货币逻辑，但避险纯度低于金。

## 四、配置定位

- **角色**：黄金对冲仓的**高 beta 卫星**。要弹性配银，要稳健配金。
- **仓位纪律**：贵金属合计 ≤5–10% 组合，SLV 因波动大占比应更小。不因回撤深视作「便宜」——无内在价值锚。
- **禁忌**：不套折扣系数、不算安全边际、不设合理价。
- **与 GLD/518680 关系**：与 GLD 同一货币逻辑、叠加工业周期；金银比是相对配置参考而非估值。三者勿重复超配。
