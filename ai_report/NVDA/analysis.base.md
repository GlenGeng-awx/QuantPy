## 五、安全边际 — 静态锚（合理价 / 满仓目标）

**新口径链条（g 不砍半；高增长封顶）：**

```
正常化 EPS = 剔投资收益后 NI ÷ 稀释股数
           ≈ 141.9B ÷ 24.46B         = $5.80   （剔 TTM 投资收益 ~$21B、税率 15.75% 还原）
           GAAP 口径                  = $6.53   （含投资收益，偏高）
g          = 第四步前瞻增速            = 远 > 22%
合理 PE    = min(8.5 + g, 30)         = 30x     （高增长硬封顶，价值框架不追）
合理价     = 5.80 × 30                = $174    （GAAP 口径 6.53 × 30 = $196）
质量判定   = 伟大 × 无麻烦（仅贵）      → 折扣系数 ×0.75
满仓目标   = 174 × 0.75               = $130    （GAAP 口径 196 × 0.75 = $147）→ $130-147
```

敏感性（取决于正常化 EPS 与是否放松封顶）：

| 情景 | 正常化 EPS | 合理 PE | 合理价 | 满仓目标（×0.75） |
|---|---|---|---|---|
| 严格（正常化 EPS $5.80，30x 封顶） | $5.80 | 30x | $174 | $130 |
| GAAP（含投资收益 $6.53，30x 封顶） | $6.53 | 30x | $196 | $147 |
| 前瞻（FY27 EPS ~$7.4，30x 封顶） | $7.40 | 30x | $222 | $167（但用未来一年利润才勉强打平，非安全边际） |

护栏检查：
- **高增长封顶**：g >> 22% → 一律 30x，价值框架不追高增长。即便如此，现价在 trailing 口径下无安全边际 → **判"买贵了"，等回调** ✓
- **回购不进 g**：g 用业务增长（本就极高），回购价值另计，不抬合理 PE ✓
- **g 质量**：`FCF − SBC = +112B`、3 年 FCF 爆发 ✓，成长质量满分——问题纯在价格不在质量
- **背离方向**：GAAP NI 含投资收益，正常化后 EPS 更低、更贵（低估贵）✓
- **净现金锚**：68B 仅 ~1.4% MCap，几乎无下行缓冲，不足以托底高估值 ✓
- **前瞻打平陷阱**：只有用 FY27 未来利润 + 30x 才勉强打平现价，即"用一年后的完美兑现来解释今天的价格"——这是买入未来、非安全边际
